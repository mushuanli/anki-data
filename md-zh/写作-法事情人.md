# 写作篇
### 写文章最重要的就是最初的100字

要想通过文章来传达自己的想法，首先要让读者愿意读下去。也就是说，我们写的文章要有足够的吸引力，这是需要一点技巧的。

写文章最重要的就是最初的100字。写在稿纸上的话，就是最初的三行文字。如果这三行文字没法抓住读者的心，那么他们就没有意愿再读下去。

下笔之初，我们应该先尝试用三行文字（100字左右）概括文章的大意。而且要学会换位思考，把自己放在读者的位置，思考怎么样的写法才能吸引读者的注意力，抓住读者的心。

我要提醒大家注意一点，一篇文章在写前三行的时候，也就是开篇明义的时候，不需要太多的写作技巧。如果把心思都花在组织语言、堆砌辞藻上，而脱离了文章的大意，那写出来的这100字，也不会吸引读者。我们必须在前三行把自己的观点、主张表达清楚。

在我们的日常工作中也是如此，想要表达的要点一定要弄清楚，不要过分重视技巧而忘记了自己最初想要表达的关键点，那样就本末倒置了。

### 一、 写作篇：用“五感”代替“结论”
**写作大忌：把话说得太满，不给读者留参与感**

我以前写东西特别喜欢自嗨，写完后，阅读的人寥寥无几。这3个月里，我才试着把所有的“我觉得”“我认为”都删掉，换成画面、声音、气味。同样的故事，数据就明显好了起来。

因为读者不关心你的结论，他们只关心自己能感受到的东西。

有一个练习方法，特别管用。下次你去咖啡馆，别着急掏出手机拍照。先坐下来，闭上眼睛，数一数你能听到几种声音：咖啡机的蒸汽声、杯碟碰撞的脆响声、邻桌女孩的谈笑声、窗外的车流声等。然后睁开眼睛，看看光线是怎么落进来的，咖啡杯上的水珠是怎么滑落的。

**这就是写作的魔法：不是告诉别人你的结论，是带着他们经历一遍你的感受。**

-   **比如你说：** 我觉得她很紧张。
-   **不如写：** 她说话的时候，声音带着明显的颤抖，话语中充满长时间的停顿和大量的空白词，“呃……那个……就是……”。

-   **再比如说：** 这包子真好吃。
-   **就写：** 刚咬一口，汤汁就淌出来了，赶紧吸，还是顺着手指往下流。肉馅紧实，带着姜的香味和一点点的甜。皮薄，被肉汁泡得有点透明，软软地贴在舌尖上。你看，我没说好吃，但你已经馋了。

这就是五感的力量。视觉、听觉、嗅觉、味觉、触觉，它们是通往读者内心的直通车。

**案例对比：**

-   **修改前：** 我小区有一个单身母亲，每天凌晨四点准时出摊卖早餐，我觉得她的背影很孤独，日复一日，看着让人心酸。
-   **修改后（重写）：**
    凌晨四点，小区门口那个早餐摊准时亮起一盏灯。她掀开帆布篷，蒸笼的热气一下子涌上来，把她整个人都裹住了。她抬手抹了一把脸，袖口已经湿了一片。
    她揉面的动作很慢，一下，一下，手指关节鼓出来，又压下去。偶尔有夜班出租车路过，车灯扫过她瘦小的背影，又移开了。
    五点差一刻，她手机响了一声。她拿面粉糊的手在围裙上蹭了蹭，低头看了看屏幕——是闹钟。她关掉闹钟，温柔地朝小区里一栋楼望了一眼——六楼那扇窗，她晾在外面的那件校服还没收。侧耳听了一下。楼上没动静。她把手机塞回兜里，接着包包子。
    第一笼包子上汽的时候，远处环卫工的扫帚声已经响了好一阵了。

这样写，我没说她孤独（不说"单身母亲"四个字，但读者看到她在意那栋楼里某个还在睡觉的人（孩子），自然就懂了。），没说我觉得，没说看着心酸。（"车灯 ... 又移开了"三个字让这个画面从单纯的环境描写变成一种隐喻——世界扫了她一眼，没有停留。）但读的人心里，自然有那个感受。

### 二、 阅读篇：用“法事情人”拆解真题，掌握答题底层逻辑

很多家长问我，什么是“法事情人”？其实很简单，包君成老师的这套方法，就是帮孩子打破“读不懂、答不对”的困境，把抽象的阅读，变成可落地、可复制的步骤，哪怕是基础薄弱的孩子，也能快速上手。

**首先要明确：** 阅读理解的标准答案，从来不是“瞎编的”，而是有固定逻辑的——**标准答案 = 文本的“事与情”（包君成「法事情人」核心）+ 命题者的“考察意图”（时代、家国、责任）**。

孩子要做的，不是“背答案”，而是通过拆解标准答案，学会“用出题人的视角看问题”，用“法事情人”的逻辑读文章。

#### 第一步：每天15-20分钟，只做1件事——拆解真题（不盲目刷题，只练真题）

不用做题，直接对照“文章+题目+标准答案”，用包君成老师 **「法事情人」四步法**拆解，每一步都落实到位：

1.  **法（结构/手法）：** 拆解文章的段落逻辑、写作手法（比喻、拟人、对比等），找到文章的“骨架”——这是答题的基础，比如概括题，就靠“法”梳理文章脉络。
2.  **事（事件/场景）：** 找出文章里的核心事件、具体场景，这是作者情感的“载体”——比如记叙文，所有的情感都藏在“事”里，答题必须紧扣事件。
3.  **情（情感/倾向）：** 透过事件，读懂作者的喜怒哀乐、价值取向——这是阅读题的“得分关键”，比如情感题、主旨题，本质就是考“情”。
4.  **人（人物/时代）：** 梳理文章中的人物形象，结合时代背景（如果有），理解人物的行为、选择——这是拔高题的“加分点”，贴合命题者的考察意图。

#### 第二步：分类梳理，掌握3大类阅读题的答题模板（包君成「事理互换」思维落地）

小初阅读理解，不管是记叙文、散文还是说明文，本质就3大类，结合包君成老师的答题逻辑，给大家整理好可直接用的思路：

1.  **人与自我（最常考）：核心是“成长、内心、品格”**
    -   **例如：** 励志故事、克服困难、个人感悟类。
    -   **答题思路：** 用「法」梳理文章结构 → 用「事」定位核心事件（前后对比，比如“以前自卑→现在自信”） → 用「情」提炼作者的情感/感悟（坚强、勇敢、诚信等） → 用「人」体现成长变化。
    -   **关键：** 答题时要突出 **“前后对比”** ，紧扣 **“成长”** ，用规范术语（比如“主旨升华”“情感递进”）。

2.  **人与自然（低年级高频，高年级拔高）：核心是“喜爱、环保、共生”**
    -   **比如写景文、写动植物的文章：**
        -   **低年级答题模板：** “本文通过描写XX（事），运用XX手法（法），表达了作者对XX（自然/家乡）的喜爱与赞美之情（情）。”
        -   **高年级拔高：** 在此基础上，加入 **“环保意识”“人与自然和谐共生”** 的理念，贴合时代要求（命题者的考察意图）。

3.  **人与社会（拔高题，分值高）：核心是“家国、文化、责任”**
    -   **比如：** 家国情怀、文化习俗、社会现象类。
    -   **答题思路：** 用「法」拆解文章逻辑 → 用「事」梳理核心场景/事件 → 用「情」体会作者的态度/观点 → 用「人」结合时代，提炼 **“家国责任”“文化传承”** 等主题。
    -   **关键：** 答题时要紧扣 **“正能量”** ，贴合主流价值取向，这是命题者最看重的点。

#### 第三步：用「事理互换」思维，规范答案表述（避免“答不到点上”）

这是包君成老师强调的核心答题思维，也是孩子最容易忽略的点：

-   **一事：** 具体的、形象的（比如文章里的事件、场景）
-   **一理：** 抽象的、概括的（比如情感、主旨、道理）

**答题时，题干问“事”，就答“理”；题干问“理”，就结合“事”来答**——比如题干问“作者为什么写这件事？”（问理），就要结合事件（事），分析作者的情感（情）和考察意图（人/时代）。

另外，答案要遵循 **“先主后次、先面后点”** 的原则，**分点作答**（基础），避免重复，语言简洁，用规范的语文术语，这样才能踩满得分点。
