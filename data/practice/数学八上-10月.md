**1. 下列说法中正确的是**

*   **解题思路要点**: 逐一分析选项。利用全等图形的定义（形状与大小均相同）及性质（面积、周长相等）进行判断。注意面积或周长相等是全等的必要不充分条件，可通过举反例排除错误选项。

*   **题目意思**: 从一个特殊四边形出发，探究其中三条线段的关系，然后逐步推广条件，探究关系是否变化。

*   **题目关键字及关键字提供的多个信息**: AB=AD, ∠EAF=1/2∠BAD, ∠B+∠ADF=180°, 延长线上的点。
*   **题目知识点**: 全等三角形的判定与性质，旋转思想，辅助线构造。
*   **解题思路要点**:
    *   核心方法是“旋转法”或构造全等。将△ABE绕点A旋转至△ADG的位置。
    *   (1) 按题目提示的思路，证明△AEF ≌ △AGF，得出EF=GF，再由GF=GD+DF=BE+DF得出结论。
    *   (2) 利用推广后的条件(∠B+∠ADF=180°)证明旋转后的G点、D点、F点共线。再利用角的关系(∠EAF=1/2∠BAD)证明△AEF ≌ △AGF，从而得出结论依然成立。
    *   (3) 点在延长线上，导致线段关系从“和”变为“差”。旋转和证明全等的思路不变，但最后一步GF的表达式变为|GD-DF|，从而得出新的结论。

#### **【题型一】全等图形概念**

**1. 下列说法中正确的是( C )**
*   **题目意思**: 从四个关于全等图形、面积、周长的命题中，选出正确的那个。
*   **题目问题**: 哪个说法是正确的？
*   **题目关键字及关键字提供的多个信息**:
    *   **全等图形**: 形状和大小都完全相同的图形。全等图形的对应边相等，对应角相等，面积相等，周长也相等。
    *   **面积相等**: 两个图形的面积数值相同。面积相等不代表形状相同，因此不一定是全等图形。
    *   **等边三角形**: 三条边都相等的三角形。所有等边三角形都相似，但边长不同则不全等。
    *   **周长相等**: 两个图形的周长数值相同。周长相等不代表形状和面积相同，因此不一定是全等图形。
*   **题目知识点**: 全等图形的定义和性质。
*   **题目问题、题目答案与关键字的联系**:
    *   A选项：面积相等的图形不一定是全等图形（如面积为4的正方形和长为4宽为1的矩形）。故A错误。
    *   B选项：两个等边三角形只有在边长相等时才是全等图形。故B错误。
    *   C选项：根据全等图形的定义，两个全等图形的大小完全相同，所以面积一定相等。故C正确。
    *   D选项：周长相等的图形不一定是全等图形（如周长为8的正方形和长为3宽为1的矩形）。故D错误。

**2. (2022 江北浦外)下列结论正确的是( D )**
*   **题目意思**: 从四个关于特定图形全等的命题中，选出正确的那个。
*   **题目问题**: 哪个结论是正确的？
*   **题目关键字及关键字提供的多个信息**:
    *   **正方形**: 四条边相等且四个角都是直角的四边形。所有正方形都相似，但不一定全等。
    *   **直角三角形**: 有一个角是直角的三角形。
    *   **有一个锐角相等**: 在两个直角三角形中，如果一个锐角对应相等，那么另一个锐角也对应相等，这两个三角形相似（AA）。但不一定全等。
    *   **两边及一个角对应相等**: 指的是SSA的情况，这不能保证两个三角形全等（特殊情况除外，如HL）。
    *   **斜边和一个锐角对应相等**: 在直角三角形中，这对应于“角角边”（AAS）判定定理，可以保证两个三角形全等。
*   **题目知识点**: 三角形全等的判定定理（SAS, ASA, AAS, SSS），直角三角形全等的判定定理（HL）。
*   **题目问题、题目答案与关键字的联系**:
    *   A选项：两个正方形只有在边长相等时才全等。故A错误。
    *   B选项：有一个锐角相等的两个直角三角形是相似的，但不一定全等。故B错误。
    *   C选项：“两边及一个角”通常指SSA，不能判定全等。故C错误。
    *   D选项：“斜边和一个锐角”对应AAS判定，可以证明两个直角三角形全等。故D正确。

**2. 如图, 在四边形ABCD 与 A'B'C'D'中...上述条件中符合要求的有( B )**
*   **题目意思**: 已知两个四边形有连续的三部分（一条边、一个角、一条边）对应相等，需要再添加一个条件使两个四边形全等。
*   **题目问题**: 在四个备选条件中，哪些可以推导出两个四边形全等？
*   **题目关键字及关键字提供的多个信息**:
    *   **四边形ABCD 与 A'B'C'D'**: 研究对象是四边形。
    *   **AB=A'B', ∠B=∠B', BC=B'C'**: 已知条件，根据SAS可判定△ABC ≌ △A'B'C'。
    *   **四边形ABCD ≌ 四边形 A'B'C'D'**: 目标。
*   **题目知识点**: 全等三角形的判定（SAS, SSS, AAS）和性质，四边形全等的判定（分解成三角形全等）。
*   **题目问题、题目答案与关键字的联系**:
    *   由已知条件AB=A'B', ∠B=∠B', BC=B'C'，可得△ABC ≌ △A'B'C'。
    *   由△ABC ≌ △A'B'C' 可得 AC = A'C', ∠BAC = ∠B'A'C', ∠BCA = ∠B'C'A'。
    *   要证四边形全等，只需再证△ADC ≌ △A'D'C'。
    *   分析添加的条件：
        *   ① ∠A=∠A', AD=A'D'：因为∠A=∠A' 且 ∠BAC=∠B'A'C'，所以∠CAD = ∠C'A'D'。加上AD=A'D' 和已证的 AC=A'C'，构成SAS，可证△ADC ≌ △A'D'C'。所以①可以。
        *   ② ∠A=∠A', CD=C'D'：同理可得∠CAD = ∠C'A'D'。但此时条件为 AC=A'C', CD=C'D', ∠CAD=∠C'A'D'，这是SSA，不能保证三角形全等。所以②不可以。
        *   ③ ∠A=∠A', ∠D=∠D'：同理可得∠CAD = ∠C'A'D'。加上∠D=∠D' 和 AC=A'C'，构成AAS，可证△ADC ≌ △A'D'C'。所以③可以。
        *   ④ AD=A'D', CD=C'D'：加上已证的AC=A'C'，构成SSS，可证△ADC ≌ △A'D'C'。所以④可以。
    *   符合要求的有①③④，所以选B。

**2. 如图, △ABC中, ∠B=∠C, ∠EDF=α...则下列结论正确的是( A )**
*   **题目意思**: 在一个等腰三角形ABC中，有两个动点D, E在底边和腰上，满足特定线段相等关系，探究一个固定角α和顶角∠A的关系。

    *   因为∠B=∠C，所以满足了SAS的判定条件。
    *   因此，△BDE ≌ △CFD (SAS)。
    *   由全等可得：∠BDE = ∠CFD。
    *   根据三角形外角性质，∠EFC = ∠EDF + ∠DEF。同时，∠EFC是△CFD的一个外角，但这不是标准用法。

    *   在△BDE中，∠DEC = 180° - ∠B - ∠BDE。
    *   点B, D, C在一条直线上，所以∠EDF + ∠BDE + ∠CDF = 180°？不，是∠BDE+∠EDC=180°。

    *   又因为∠EDC = ∠EDF + ∠CDF = α + ∠CDF。
    *   所以 α + ∠CDF = 180° - ∠B - ∠CDF。这个推导似乎有问题。
    *   **正确思路**:
        *   证明 △BDE ≌ △CFD (SAS)。已知 BE=CD, BD=CF, ∠B=∠C。
        *   所以 ∠BED = ∠CDF。
        *   因为 ∠CED = 180° - ∠BED - ∠C。 (在△EFC中，∠FEC=180-∠EFC-∠C)
        *   由外角性质，对于△BDE, ∠ADC = ∠B + ∠BED。
        *   又 ∠ADC = ∠ADE + ∠EDC.
        *   由外角性质，对于△DEF, ∠CEF = ∠EDF+∠DFE
        *   由外角性质，对于△CDF, ∠EFD = ∠C + ∠CDF。
        *   Let's use ∠BDE = ∠CFD.  ∠A + ∠B + ∠C = 180°. 因为∠B=∠C, 所以 ∠A + 2∠B = 180°。
        *   ∠DEF = 180° - ∠EDF - ∠DFE = 180° - α - ∠DFE.
        *   在直线BC上，∠BDE + ∠EDF + ∠FDC = 180°. (如果E,D,F在一条直线上)
        *   **最简思路**:
            *   ∠CED 是 △BDE 的一个外角，所以 ∠CED = ∠B + ∠BDE。
            *   在 △CDE 中，∠C + ∠CED + ∠CDE = 180°。
            *   ∠C + (∠B + ∠BDE) + ∠CDE = 180°。
            *   因为 ∠B=∠C，∠CDE = ∠CDF + α。
            *   所以 ∠B + (∠B + ∠BDE) + (∠CDF + α) = 180°。
            *   由全等△BDE ≌ △CFD 可知 ∠BDE = ∠CFD。这个代换没用。
            *   应该是 ∠BED = ∠CDF。
            *   ∠EDC = 180° - ∠BDE - ∠B。不对。

            *   而 ∠EDC = ∠EDF + ∠FDC = α + ∠FDC。
            *   所以 α + ∠FDC = ∠B + ∠BED。
            *   因为 △BDE ≌ △CFD，所以 ∠BED = ∠CDF。
            *   代入得：α + ∠FDC = ∠B + ∠CDF。
            *   所以 α = ∠B。
            *   又因为 △ABC 是等腰三角形，∠A + ∠B + ∠C = 180°，即 ∠A + 2∠B = 180°。

*   **题目问题、题目答案与关键字的联系**:
