### **【模块一】三角形全等**

---

#### **1. 题目一**
*   **题目意思**：△ABC中，D, E, F为边上点，已知`∠BAC = α`, `∠FED = θ`，且`△BED ≅ △CFE`。
*   **题目问题**：求α与θ的关系。
*   **关键字**：`△BED ≅ △CFE`，点E在BC上。
*   **知识点**：全等三角形性质，三角形内角和，平角定义。
*   **解题思路要点**：
    1.  利用全等得出 `∠B = ∠C` (△ABC为等腰) 和 `∠BDE = ∠CEF`。
    2.  利用平角 `∠BED + ∠DEF + ∠CEF = 180°` 进行角度代换。
    3.  在△BDE中使用内角和定理，对比等式得出 `∠B = θ`。
    4.  在△ABC中使用内角和定理，建立α与θ的关系。

---

#### **2. 题目二**
*   **题目意思**：在△ABC和△ADC中，已知`∠BAC = ∠DAC`，有公共边AC。
*   **题目问题**：分别按SAS, AAS, ASA判定，添加一个条件使△ABC ≅ △ADC。
*   **关键字**：SAS, AAS, ASA，公共边。
*   **知识点**：三角形全等判定公理。
*   **解题思路要点**：
    1.  **SAS**：已有`A(角)`和`S(边AC)`，需补充夹`A`的另一边，即`AB=AD`。
    2.  **AAS**：已有`A(角)`和`S(边AC)`，需补充不与`S`相邻的另一对角，即`∠B=∠D`。
    3.  **ASA**：已有`A(角)`和`S(边AC)`，需补充与`S`相邻的另一对角，即`∠BCA=∠DCA`。

---

#### **3. 题目三**
*   **题目意思**：四边形ABCD中，`∠B=∠C`，E是AB中点，动点P在BC上，动点Q在CD上。
*   **题目问题**：当△BPE与△PCQ全等时，求Q点的速度。
*   **关键字**：动点问题，全等。
*   **知识点**：全等三角形判定(SAS)，动点问题方程思想，分类讨论。
*   **解题思路要点**：
    1.  用时间`t`和速度`v`表示线段`BP`, `PC`, `CQ`。
    2.  `BE`长度已知。
    3.  根据`∠B=∠C`，应用SAS判定全等，存在两种对应情况。
    4.  **Case 1: △BPE ≅ △PCQ**，列出方程组 `BP=PC`, `BE=CQ`，求解`v`。
    5.  **Case 2: △BPE ≅ △CQP**，列出方程组 `BP=CQ`, `BE=CP`，求解`v`。

#### **1. 题目一**

*   **题目意思**：在一个三角形ABC中，D、E、F分别是三条边AB、BC、CA上的点。已知∠BAC = α，∠FED = θ，并且三角形BED和三角形CFE是全等的。
*   **题目问题**：找出角度α和θ之间满足的关系式。
*   **题目关键字及关键字提供的多个信息**：
    *   **△BED ≅ △CFE (三角形全等)**：这是核心条件。它提供了以下信息：
        1.  **对应角相等**：∠B = ∠FCE (即∠C)，∠BED = ∠CFE，∠BDE = ∠CEF。
        2.  **对应边相等**：BE = CF，ED = FE，BD = CE。
    *   **点E在BC上**：这意味着∠BED, ∠DEF, ∠CEF 三个角在一条直线上，它们的和是180°。
*   **题目知识点**：
    1.  全等三角形的性质。
    2.  三角形内角和定理。
    3.  平角的定义。
    4.  等腰三角形的判定与性质。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  从全等条件 `△BED ≅ △CFE` 出发，得到 `∠B = ∠C` 和 `∠BDE = ∠CEF`。
    2.  `∠B = ∠C` 说明 `△ABC` 是一个等腰三角形。
    3.  因为点E在直线BC上，所以 `∠BED + ∠DEF + ∠CEF = 180°`。
    4.  将已知 `∠DEF = θ` 和从全等得到的 `∠CEF = ∠BDE` 代入上式，得到 `∠BED + θ + ∠BDE = 180°`。
    5.  在△BDE中，根据三角形内角和定理，`∠B + ∠BED + ∠BDE = 180°`。
    6.  比较步骤4和5的两个等式，可以推出 `∠B = θ`。
    7.  在等腰△ABC中，`∠BAC + ∠B + ∠C = 180°`。将 `∠BAC = α`, `∠B = θ`, `∠C = θ` 代入，得到 `α + θ + θ = 180°`，即 `α + 2θ = 180°`。

#### **2. 题目二**

*   **题目意思**：在△ABC和△ADC中，已知∠BAC = ∠DAC，并且它们共享一条边AC。需要在不添加辅助线的前提下，为“SAS”、“AAS”、“ASA”三种全等判定方法各添加一个条件，使得△ABC ≅ △ADC。
*   **题目问题**：根据三种不同的全等判定方法，分别填写所需的条件。
*   **题目关键字及关键字提供的多个信息**：
    *   **∠BAC = ∠DAC**：提供了一对相等的角 (A)。
    *   **公共边AC**：天然存在一对相等的边 (S)，即 `AC = AC`。
    *   **SAS, AAS, ASA**：三种不同的判定公理，它们规定了边和角必须满足的相对位置关系。
*   **题目知识点**：
    1.  三角形全等的判定公理：SAS（边角边），AAS（角角边），ASA（角边角）。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  **(1) SAS (边角边)**：已知“边AC”和“角∠BAC”。SAS要求角是两边的夹角。已知的角 `∠BAC` 是边 `AB` 和边 `AC` 的夹角。因此，需要添加另一条边相等，即 `AB = AD`。
    2.  **(2) AAS (角角边)**：已知“角∠BAC”和“边AC”。AAS要求边不是两个角的夹边。边AC的邻角是∠BAC和∠BCA。为了满足AAS，需要添加另一对角相等，但这对角不能是边AC的另一个邻角。所以应添加 `∠B = ∠D`。
    3.  **(3) ASA (角边角)**：已知“角∠BAC”和“边AC”。ASA要求边是两个角的夹边。已知的边 `AC` 的邻角是 `∠BAC` 和 `∠BCA`。因此，需要添加另一对邻角相等，即 `∠BCA = ∠DCA`。

#### **1. 题目一**

*   **题目意思**：已知两个等边三角形△ABC和△BPE，点P在BC的延长线上，EC的延长线与AP相交于点M。给出四个关于图中线段和角度的结论。

*   **题目关键字及关键字提供的多个信息**：

    *   **P在BC的延长线上**：表示B、C、P三点共线，且顺序为B-C-P。这可以确定一些关键角度。
    *   **旋转模型**：两个共顶点的等边三角形（或等腰直角三角形等）是典型的旋转模型标志。
*   **题目知识点**：
    1.  全等三角形的判定与性质（SAS）。
    2.  几何旋转思想。
    3.  圆内接四边形的性质。
    4.  托勒密定理（Ptolemy's theorem）。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  **证明△ABP ≅ △CBE (SAS)**：
        *   边：`BA = BC` (△ABC等边), `BP = BE` (△BPE等边)。
        *   角：因为B-C-P共线，所以`∠ABC = 60°`。`∠PBE = 60°`。`∠ABP`和`∠CBE`是什么关系？因为`∠ABC=∠PBE=60°`, 所以 `∠ABC+∠CBE = ∠PBE+∠CBE` ? 不对。应该是 `∠ABP = ∠ABC = 60°` 因为P在BC延长线上。`∠CBE` 也是60°。因此 `∠ABP = ∠CBE`。
        *   因此△ABP ≅ △CBE。
    2.  **分析结论**：
        *   **① AP = CE**：由△ABP ≅ △CBE直接得出，正确。
        *   **② ∠PME = 60°**：因为△ABP可由△CBE绕点B旋转60°得到（或反之），所以线段AP和CE的夹角等于旋转角60°。正确。
        *   **③ BM 平分 ∠AME**：需要证明∠AMB = ∠EMB。
            *   由△ABP ≅ △CBE，得 `∠BAP = ∠BCE`。
            *   这个角相等条件 (`∠MAB = ∠MCB`) 证明了A, B, C, M四点共圆。
            *   因为A, B, C, M共圆，所以∠AMB和∠ACB是同弧（AB）所对的圆周角，因此`∠AMB = ∠ACB`。
            *   又因△ABC是等边三角形，`∠ACB = 60°`。所以`∠AMB = 60°`。
            *   因为∠PME = 60°，所以其对顶角∠AMC=60°。那么∠AME = 180° - ∠AMC = 120°。
            *   所以 `∠EMB = ∠AME - ∠AMB = 120° - 60° = 60°`。
            *   因此 `∠AMB = ∠EMB`，结论正确。
        *   **④ AM + MC = BM**：

    *   `AD=BD`, `DE=CE`, `△ADE` is isosceles：提供了三组等腰三角形的条件。
*   **题目知识点**：
    1.  等腰三角形的性质。
    2.  三角形内角和与外角定理。
    3.  分类讨论。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  由`AD=BD`和`∠B=30°`可知，`△ABD`是等腰三角形。
    2.  `∠BAD = ∠B = 30°`。`∠ADB = 180° - 30° - 30° = 120°`。
    3.  `∠ADC = 180° - ∠ADB = 180° - 120° = 60°`。
    4.  设`∠C = x`。由`DE=CE`可知`△DEC`是等腰三角形，`∠EDC = ∠C = x`。
    5.  `∠ADE = ∠ADC - ∠EDC = 60° - x`。
    6.  `∠DEC = 180° - 2x`。
    7.  `∠AEC`是`△BEC`的外角，`∠AEC = ∠B + ∠BCE = 30° + x`? 不对，E不在BC延长线上。
    8.  在`△ABC`中，`∠BAC = 180° - ∠B - ∠C = 180° - 30° - x = 150° - x`。
    9.  `∠DAE = ∠BAC - ∠BAD = (150° - x) - 30° = 120° - x`。
    10. `∠AED`是`△AEC`中的角，也是`∠DEC`的补角？不一定。`∠AED = 180 - ∠DEC - ∠AE*`。
    11. 在`△ADE`中，三个角分别为 `∠DAE=120°-x`，`∠ADE=60°-x`，`∠AED = 180° - (120°-x) - (60°-x) = 2x`。
    12. 已知`△ADE`是等腰三角形，对其三边相等关系进行分类讨论：
        *   **Case 1: AE = AD** -> `∠AED = ∠ADE` -> `2x = 60° - x` -> `3x = 60°` -> `x = 20°`。
        *   **Case 2: DE = DA** -> `∠DEA = ∠DAE` -> `2x = 120° - x` -> `3x = 120°` -> `x = 40°`。
        *   **Case 3: ED = EA** -> `∠EDA = ∠EAD` -> `60° - x = 120° - x` -> `60° = 120°`，此情况不可能。
    13. 因此，`∠C`的度数可能为 **20°** 或 **40°**。通常这类填空题有一个默认的答案，但根据条件两种都可能。

#### **Page 11: Problem 7**

*   **题目意思**：在一条直线PQ上有一个定点O，直线外有一个定点A。
*   **题目问题**：在直线PQ上找点B，使得`△AOB`是等腰三角形，问这样的点B最多有几个。
*   **题目关键字及关键字提供的多个信息**：
    *   **等腰三角形AOB**：`AO=AB` 或 `BO=BA` 或 `OA=OB`。
    *   **点B在直线PQ上**：点B的轨迹是一条直线。
*   **题目知识点**：
    1.  圆的定义（到定点的距离等于定长的点的集合）。
    2.  线段垂直平分线的定义（到线段两端点距离相等的点的集合）。
    3.  直线与圆、直线与直线的位置关系。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    *   分三种情况讨论：
        1.  **OA = OB**：点B在以O为圆心，OA为半径的圆上。此圆与直线PQ相交，最多产生 **2** 个交点B。
        2.  **AO = AB**：点B在以A为圆心，AO为半径的圆上。此圆与直线PQ相交，最多产生 **2** 个交点B（其中一个点是O本身，但题目要求构成三角形，所以O点不算。如果B与O重合，则无法构成三角形。但题目问的是点的个数，所以要看具体要求。通常情况下，B不能与O重合。所以最多产生 **1** 个新的点B）。
        3.  **BO = BA**：点B到点O和点A的距离相等。点B的轨迹是线段OA的垂直平分线。这条垂直平分线与直线PQ相交，最多产生 **1** 个交点B。（除非OA的垂直平分线与PQ平行，但题目图示为一般情况）。
    *   **合计**：将三种情况下的点B加起来，`2 + 1 + 1 = 4`。
    *   所以，最多可以找到 **4** 个这样的点B。
