### **第一页**

#### **1. 选择题**
*   **题目意思**：已知两个全等三角形△ABC和△DEF，其中四个顶点F, B, E, C在一条直线上。给出两个角的度数，求另一个角的度数。
*   **题目问题**：求∠DEC的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `△ABC ≅ △DEF`：全等三角形。这意味着对应角相等（∠A=∠D, ∠ABC=∠DEF, ∠ACB=∠DFE），对应边相等。
    *   `点F, B, E, C在同一条直线上`：共线。这意味着以E为顶点的角∠DEF和∠DEC是邻补角，它们的和为180°。
    *   `∠A=34°`, `∠F=36°`：已知的角度。
*   **题目知识点**：全等三角形的性质，三角形内角和定理，邻补角的定义。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  利用`全等`性质，由∠A=34°可知对应角∠D=34°。∠F即∠DFE，为36°。
    2.  在△DEF中，利用`三角形内角和定理`（180°），求出∠DEF的度数。
    3.  利用`共线`条件，可知∠DEF与∠DEC互为邻补角，相加为180°，从而求出∠DEC。

#### **2. 选择题**
*   **题目意思**：在一个三角形草坪内找一个点建凉亭。
*   **题目问题**：凉亭的位置需要满足“到草坪三条边的距离相等”。
*   **题目关键字及关键字提供的多个信息**：
    *   `到三条边的距离相等`：这是角平分线交点的核心性质。到一个角两边距离相等的点在该角的平分线上。因此，到三条边距离相等的点是三条角平分线的交点。
*   **题目知识点**：三角形的“四心”定义和性质（内心、外心、重心、垂心）。
    *   内心：三条角平分线的交点，到三边距离相等。
    *   外心：三条边垂直平分线的交点，到三顶点距离相等。
    *   重心：三条中线的交点。
    *   垂心：三条高的交点。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  将问题中的`到三条边的距离相等`与三角形“四心”的性质进行匹配。
    2.  确定这一定义对应的是`三条角平分线的交点`（内心）。

#### **3. 选择题**
*   **题目意思**：在△ABC中，已知∠B=∠C，以及另外三个条件∠EDF=α, BD=CF, BE=CD。
*   **题目问题**：找出α和∠A之间的正确关系式。
*   **题目关键字及关键字提供的多个信息**：
    *   `∠B=∠C`：△ABC是等腰三角形。
    *   `BD=CF, BE=CD`：两组边相等。
    *   `∠EDF=α`：一个固定角度。
*   **题目知识点**：全等三角形的判定（SAS），三角形外角的性质，等腰三角形的性质。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  观察△BDE和△CFD。已知`BE=CD`，`BD=CF`，`∠B=∠C`。
    2.  利用`SAS`证明△BDE ≅ △CFD。
    3.  由全等可知对应角相等，即∠BDE = ∠CFD。
    4.  利用平角或三角形外角性质建立角之间的关系。例如，∠ADC是△BDE的外角，∠ADC=∠B+∠BED。同时，∠EDF+∠FDC=∠EDC。通过这些角度的转换，将α和∠A联系起来。一个常用的技巧是利用∠DEC = 180° - ∠BED - ∠CFD，并把∠CFD换成∠BDE，从而建立∠B和α的关系，再结合△ABC内角和，最终得到α和∠A的关系。

#### **5. 选择题**

*   **题目关键字及关键字提供的多个信息**：

    3.  **情况二：夹角是钝角。** 这意味着顶角是钝角。高线画在三角形外部。高线、另一腰的延长线和底边形成一个直角三角形。通过角度关系计算，可以发现顶角与这个45°角的关系，最终求出顶角。

#### **6. 选择题**
*   **题目意思**：在一个面积已知的等腰三角形ABC中，D是底边BC上一点，作了两条高DE和DF。
*   **题目问题**：已知DF=3DE，求DF的长度。
*   **题目关键字及关键字提供的多个信息**：
    *   `△ABC的面积为48`：已知面积。
    *   `AB=AC=6`：等腰三角形，腰长为6。
    *   `DE⊥AB, DF⊥AC`：DE和DF是点D到两腰的距离。
    *   `DF=3DE`：两条高之间的关系。
*   **题目知识点**：三角形面积公式，等腰三角形的性质。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  连接AD。`△ABC的面积`可以被AD分成`△ABD的面积`和`△ACD的面积`之和。
    2.  S△ABC = S△ABD + S△ACD。
    3.  利用面积公式：S△ABD = (1/2) * AB * DE，S△ACD = (1/2) * AC * DF。
    4.  代入已知数据：48 = (1/2) * 6 * DE + (1/2) * 6 * DF。
    5.  简化得到方程：48 = 3 * DE + 3 * DF，即 DE + DF = 16。
    6.  结合已知条件 `DF=3DE`，得到一个二元一次方程组，解出DE和DF的值。

#### **7. 选择题**
*   **题目意思**：在△ABC中，P是内部一点。点M在PA的垂直平分线上，点N在PC的垂直平分线上。
*   **题目问题**：求∠APC的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `M在PA的垂直平分线上`：垂直平分线的性质，线段上任意一点到线段两端点距离相等。所以 MA = MP。
    *   `N在PC的垂直平分线上`：同理，NC = NP。
    *   `∠ABC=52°`：已知角。
*   **题目知识点**：线段垂直平分线的性质，等腰三角形的性质（等边对等角），三角形内角和定理。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  由`M在PA的垂直平分线上`得到MA=MP，从而△MAP是等腰三角形，所以∠MAP = ∠MPA。
    2.  由`N在PC的垂直平分线上`得到NC=NP，从而△NCP是等腰三角形，所以∠NCP = ∠NPC。
    3.  ∠APC = ∠MPA + ∠NPC。所以 ∠APC = ∠MAP + ∠NCP = ∠CAB + ∠ACB。
    4.  在△ABC中，利用`三角形内角和定理`，∠CAB + ∠ACB = 180° - ∠ABC。
    5.  代入已知`∠ABC=52°`，即可求出∠APC。

#### **8. 选择题**
*   **题目意思**：在△ABC中，∠A=60°，两条角平分线BD和CE相交于点O。已知周长和BC边长，以及AE:AD=4:3。

*   **题目知识点**：角平分线定理，三角形内角和定理，等边三角形的判定和性质。
*   **解题思路**：
    1.  在△ABC中，由∠A=60°和BD、CE是角平分线，可求出∠BOC=120°。
    2.  证明△AOE和△AOD中的一些关系，或者利用角平分线定理。
    3.  利用`角平分线定理`：在△ABC中，BD平分∠ABC，则 AD/DC = AB/BC。CE平分∠ACB，则 AE/EB = AC/BC。
    4.  证明△AEF和△ADG（其中F，G在BC上）全等或者相似，但这题可能更复杂。
    5.  一个关键点是在AB上取点F使AF=AD，证明△ADF为等边三角形，再证明△AFC≅△ADB，导出AF=AC-BC，从而建立边的关系。这题方法较多，但都围绕角平分线和等腰/等边三角形展开。

#### **9. 填空题**
*   **题目意思**：一个等腰三角形的一个外角是100°。
*   **题目问题**：求顶角的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `等腰三角形`：两底角相等。
    *   `外角度数为100°`：外角和与之相邻的内角互补（和为180°）。外角等于不相邻的两个内角之和。需要分类讨论这个外角是顶角的外角还是底角的外角。
*   **题目知识点**：等腰三角形的性质，三角形外角的性质，三角形内角和定理。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  **情况一：100°是底角的外角。** 那么这个底角为 180°-100°=80°。因为是等腰三角形，另一个底角也是80°。顶角 = 180°-80°-80° = 20°。
    2.  **情况二：100°是顶角的外角。** 那么顶角为 180°-100°=80°。两个底角和为100°，每个底角为50°。这两种情况都成立。题目问顶角，所以答案是20°或80°。

#### **10. 填空题**
*   **题目意思**：在直角△ABC中，E是斜边AB的中点，D在BC上，且AD=BD。
*   **题目问题**：求∠DFE的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `∠ACB=90°`：直角三角形。
    *   `E为AB的中点`：CE是斜边上的中线。性质：斜边上的中线等于斜边的一半（CE=AE=BE）。
    *   `AD=BD`：△ABD是等腰三角形。
    *   `∠B=20°`：已知角。
*   **题目知识点**：直角三角形斜边中线性质，等腰三角形性质，三角形外角性质。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  由`E是AB中点`和`∠ACB=90°`，得CE=BE，所以△BCE是等腰三角形，∠BCE = ∠B = 20°。
    2.  由`AD=BD`，得△ABD是等腰三角形，∠BAD = ∠B = 20°。
    3.  在△ABC中，∠A=90°-∠B=70°。
    4.  ∠DFE是△ADF的外角，或在△FDC中计算。考虑△AFC，∠FAC=∠A=70°，∠FCA=∠ACB-∠BCE=90°-20°=70°。所以△AFC是等腰三角形，AF=CF。
    5.  ∠AFC = 180°-70°-70°=40°。
    6.  ∠DFE与∠AFC是对顶角，所以∠DFE=∠AFC=40°。

#### **11. 填空题**
*   **题目意思**：两条角平分线交于E，过E作BC的平行线MN。
*   **题目问题**：求MN的长度。
*   **题目关键字及关键字提供的多个信息**：
    *   `∠ABC和∠ACB的平分线交于点E`：BE平分∠ABC，CE平分∠ACB。
    *   `MN // BC`：平行线。性质：内错角相等。
*   **题目知识点**：角平分线的定义，平行线的性质，等腰三角形的判定（等角对等边）。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  因为`MN // BC`，所以∠MEB = ∠EBC（内错角相等）。
    2.  因为`BE是角平分线`，所以∠MBE = ∠EBC。
    3.  因此，∠MEB = ∠MBE，所以△MBE是等腰三角形，ME = MB。
    4.  同理可证，∠NEC = ∠ECB，∠NCE = ∠ECB，所以∠NEC = ∠NCE，△NCE是等腰三角形，NE = NC。
    5.  MN = ME + NE。将ME和NE替换为MB和NC，即可求出MN的长度。

#### **12. 填空题**
*   **题目意思**：AD是△ABC的中线。
*   **题目问题**：求中线AD的取值范围。
*   **题目关键字及关键字提供的多个信息**：
    *   `中线`：连接顶点和对边中点的线段。
    *   `AB=12, AC=8`：已知两边长。
*   **题目知识点**：三角形三边关系定理（两边之和大于第三边，两边之差小于第三边），构造全等三角形（倍长中线法）。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  使用“倍长中线法”。延长AD到E，使得DE=AD，连接BE。
    2.  证明△ADC ≅ △EDB（利用SAS，因为AD=ED，∠ADC=∠EDB，CD=BD）。
    3.  由全等可知，BE = AC = 8。
    4.  在△ABE中，三边为AB=12, BE=8, AE=2AD。
    5.  应用`三角形三边关系定理`：AB - BE < AE < AB + BE。
    6.  即 12 - 8 < 2AD < 12 + 8，解出AD的范围。

#### **13. 填空题**
*   **题目意思**：在四边形ABCD中，CB=CD，AC平分∠BAD。
*   **题目问题**：求∠B的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `CB=CD`：△BCD是等腰三角形。
    *   `AC平分∠BAD`：∠BAC = ∠DAC。
    *   `∠ACB=90°, ∠ACD=42°`：已知角度。
*   **题目知识点**：全等三角形的判定（SAS），角的和差计算。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  在△ABC和△ADC中，我们有∠BAC = ∠DAC，AC=AC，但是边CB和CD不是夹边，不能直接用SAS。
    2.  考虑过点C作CA的垂线，或者利用其他方法。一个简单的方法是过点C作CE⊥AD于E，CF⊥AB于F。由角平分线性质可知CE=CF。
    3.  在Rt△BFC和Rt△DEC中，CF=CE，CB=CD。根据HL可以证明Rt△BFC ≅ Rt△DEC。
    4.  由此得到∠B=∠CDE。
    5.  在△ABC中，∠B = 90° - ∠BAC。在△ADC中，利用内角和求出∠ADC。通过这些关系建立方程求解。
    6.  或者，在△ABC和△ADC中，我们有AC，有CB=CD，有∠BAC = ∠DAC。这不是标准的判定方法。但可以尝试用正弦定理（超纲）或者构造全等。
    7.  最直接的方法：在AB上取一点E，使得AE=AD。连接CE。可证明△AEC≅△ADC(SAS)。所以CE=CD，∠ACE=∠ACD=42°。因为CB=CD，所以CE=CB。△BCE是等腰三角形。∠CEB=∠B。∠ACB=90°，所以∠BCE=90°-42°=48°。在△BCE中，2∠B + 48° = 180°，解出∠B。

#### **14. 填空题**
*   **题目意思**：点O是△ABC外一点（或内），且OA=OB=OC。
*   **题目问题**：求∠OCA的度数。
*   **题目关键字及关键字提供的多个信息**：
    *   `OA=OB=OC`：点O是△ABC的外心（三边垂直平分线的交点）。这同时说明△OAB, △OBC, △OAC都是等腰三角形。

*   **题目问题、题目答案与关键字的联系（解题思路）**：
    1.  因为`OA=OB`，所以△OAB是等腰三角形，∠OBA = ∠OAB = 20°。
    2.  因为`OB=OC`，所以△OBC是等腰三角形，∠OCB = ∠OBC = 30°。

#### **15. 填空题**
*   **题目意思**：D是△ABC内一点，CD平分∠ACB，BD⊥CD，∠A=∠ABD。
*   **题目问题**：求AC的长。
*   **题目关键字及关键字提供的多个信息**：
    *   `CD平分∠ACB`：∠ACD = ∠BCD。
    *   `BD⊥CD`：∠BDC=90°。
    *   `∠A=∠ABD`：△ABD是等腰三角形，AD=BD。
    *   `BD=1, BC=3`：已知边长。
*   **题目知识点**：角平分线性质，等腰三角形的判定与性质，构造全等三角形。
*   **题目问题、题目答案与关键字的联系（解题思路）**：

#### **26. 解答题**
*   **题目意思**：将一个直角三角形纸片折叠，使顶点A落在直角边BC上。
*   **题目问题**：(1) 已知∠AFE=65°，求∠CDF。(2) 若△CDF和△BDE都是等腰，求∠B。
*   **题目关键字及关键字提供的多个信息**：
    *   `折叠`：折叠前后的图形关于折痕对称，是全等形。即△AFE ≅ △DFE。
    *   `∠ACB=90°`：直角三角形。
*   **题目知识点**：折叠的性质，全等三角形的性质，平行线性质，等腰三角形的性质，分类讨论。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    *   **(1) 求∠CDF**
        1.  由`折叠`可知，∠AFE = ∠DFE = 65°，∠A = ∠EDF。
        2.  因为AF和DF在AC上，所以∠AFC=180°，所以∠EFC=180-65-65=50°。
        3.  在Rt△FCD中，∠C=90°，所以∠CFD = 90°。在Rt△FCD中，∠FDC=90°-∠EFC=90-50=40°。
        4.  AF//CD? 不一定。
        5.  思路2：由折叠，∠AEF = ∠DEF。EF是折痕。AF=DF。∠A=∠ADF。
        6.  在△AFE中，∠A = 180° - ∠AFE - ∠AEF = 180 - 65 - ∠AEF。
        7.  ∠CDF与∠A有关。∠CDF = 180 - ∠ADF - ∠BDE。
    *   **(2) 求∠B**
        1.  需要对△CDF和△BDE的等腰情况进行`分类讨论`。
        2.  由折叠，AF=DF。在△CDF中，CD和DF可能相等，CD和CF可能相等，DF和CF可能相等。
        3.  同时△BDE也等腰，BD和DE，BD和BE，DE和BE可能相等。
        4.  将这些情况组合，利用角度关系（∠B+∠A=90°，折叠的角相等）列方程求解。例如，若CF=DF，则DF=AF，则CF=AF，所以F是AC中点。再讨论△BDE的情况。这是一个非常复杂的分类讨论题。

#### **27. 解答题**
*   **题目意思**：在一个等腰△DEF中，B在底边EF上。C是射线BD上一点，且在BE上截取BA=BC。
*   **题目问题**：探究线段AE, BF, CD之间的数量关系。
*   **题目关键字及关键字提供的多个信息**：
    *   `DE=DF`：△DEF是等腰三角形。
    *   `∠EBD=60°`：特殊角。
    *   `BA=BC`：△ABC是等腰三角形。
*   **题目知识点**：等腰三角形性质，等边三角形判定与性质，全等三角形的构造与证明（通常是旋转）。
*   **题目问题、题目答案与关键字的联系（解题思路）**：
    *   **(1) ① C与D重合**
        1.  补全图形，此时C就是D，BA=BD。
        2.  在△ABE和△DBF中，寻找全等关系。
        3.  因为DE=DF，所以∠E=∠F。
        4.  已知BA=BD，但无法直接证明全等。
        5.  考虑旋转。将△ABD绕点B旋转60°，A点会与C点（即D点）重合吗？不一定。
        6.  关键在于`∠EBD=60°`和`BA=BC`。如果∠ABC=60°，则△ABC是等边。
        7.  将△ABE绕点B逆时针旋转60°，得到△CBG。则AE=CG，∠E=∠BCG。再证明△BFD和△CGD的关系。
        8.  最终关系通常是和或差，如AE=BF+CD。
    *   **(1) ② C在BD上**
        1.  证明 `AE=BF+CD`。
        2.  使用`旋转法`。将△ABC绕点B逆时针旋转60°得到△A'BC'。A'会和E重合吗？
        3.  标准思路：在BF上截取BG=CD，连接... 或者延长...
        4.  旋转△ABE绕点B 60°得到△CBG。则AE=CG, ∠BAE=∠BCG。点G在△CBF内部或外部。
        5.  由于∠ABC=180-2∠BAC，不确定。
        6.  正确思路：在BF上截取BM=AE。或者在AE上...
        7.  经典模型：将△ABE绕B点顺时针旋转60°，得到△CB'E'。E'在BF上，AE=CE'。然后证明CE'=BF+CD。
    *   **(2) C在BD延长线上**
        1.  此时C在D的另一侧。
        2.  关系可能变为差或另一种和。例如 `AE = BF - CD` 或 `BF = AE + CD`。
        3.  方法同上，通过旋转构造全等，然后观察线段的和差关系。
