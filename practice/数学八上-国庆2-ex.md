### **第一部分：选择题**

#### **第1题**
*   **题目意思**: 在一个三角形中，最长的一条边所对应的那个角是一个锐角。
*   **题目问题**: 判断这个三角形的形状。
*   **题目关键字及关键字提供的多个信息**:
    *   **最长的边**: 三角形中，大边对大角，所以这个三角形中最大的角就是这个最长边所对的角。
    *   **锐角**: 角度小于90度。
*   **题目知识点**: 三角形内角和定理，三角形边与角的关系（大边对大角），三角形按角分类（锐角、直角、钝角三角形）。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  利用“大边对大角”的性质，确定最长边所对的角是三角形中最大的角。
    2.  题目告知这个最大的角是锐角。
    3.  根据三角形分类的定义，一个三角形中最大的角都是锐角，那么其它两个角必然也是锐角，因此这个三角形一定是锐角三角形。

#### **第2题**
*   **题目意思**: 用四根长度分别为1, 1, 3, x的线段，首尾相接构成一个凸四边形。
*   **题目问题**: x的值可能是四个选项中的哪一个。
*   **题目关键字及关键字提供的多个信息**:
    *   **凸四边形**: 四个内角都小于180度的四边形。
    *   **首尾顺次相接**: 意味着这四条边能构成一个封闭图形。
*   **题目知识点**: 三角形三边关系定理的推广（在四边形中，任意一条边的长度必须小于另外三条边的长度之和）。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  应用四边形的三边关系：任意一边小于其他三边之和。
    2.  将x和已知的三边代入不等式。最关键的不等式是“最长边 < 其余三边之和”。
    3.  分情况讨论：如果x不是最长边，则最长边是3，需满足 3 < 1 + 1 + x，解得 x > 1。
    4.  如果x是最长边，则需满足 x < 1 + 1 + 3，解得 x < 5。
    5.  综合得到x的取值范围是 1 < x < 5。
    6.  在选项A, B, C, D中选择符合此范围的值。

#### **第3题**
*   **题目意思**: 在等腰三角形ABC (AC=BC)中，点E在射线BF上，已知∠CBF = ∠CAE。CD垂直于BF，已知DE和AE的长度。
*   **题目问题**: 求BD的长度。
*   **题目关键字及关键字提供的多个信息**:
    *   **AC=BC**: △ABC是等腰三角形，意味着∠CAB = ∠CBA。
    *   **∠CBF = ∠CAE**: 一个关键的角度相等条件，通常用于构造全等三角形。
    *   **CD ⊥ BF**: 构造了直角三角形△CDB，CD是高。
    *   **DE=1, AE=4**: 已知的线段长度。
*   **题目知识点**: 等腰三角形的性质，全等三角形的判定与性质（特别是ASA或AAS）。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  目标是求BD。BD在Rt△CDB中，但条件不足以直接求解。思路转向利用全等变换。
    2.  利用AC=BC, ∠CAE=∠CBF, 以及由AC=BC推导出的∠CAB=∠CBA，可以尝试构造全等三角形。
    3.  一个常见的辅助线作法是：在AE上或其延长线上取一点G，使得△BCF ≅ △ACG，或者旋转△CAE。
    4.  更简洁的思路是：构造△BCG与△ACE全等。在射线BF上取点G，使得BG=AE=4，连接CG。证明△CBG ≅ △CAE (SAS)。
    5.  由全等可知CE=CG，∠BGC = ∠AEC。
    6.  在等腰△CEG中，CD是高，所以D是底边EG的中点。
    7.  因此 ED = DG = 1。
    8.  BD = BG - DG，代入已知和求得的值即可。

#### **第4题**
*   **题目意思**: 在一个已知的等腰三角形ABC中，底边和面积已知。EF是腰AC的垂直平分线。D是底边中点。M是EF上的一个动点。
*   **题目问题**: 求△CDM周长的最小值。
*   **题目关键字及关键字提供的多个信息**:
    *   **等腰△ABC**: AB=AC。
    *   **BC=4cm, 面积=16cm²**: 可以用来求出三角形的高。
    *   **EF是AC的垂直平分线**: EF上的任意一点到A和C的距离相等（MA=MC）。
    *   **D是BC的中点**: BD=DC=2。
    *   **M是直线EF上的动点**: M的位置不固定。
    *   **周长的最小值**: 典型的“将军饮马”问题，利用轴对称求最短路径。
*   **题目知识点**: 轴对称求最短路径，线段垂直平分线的性质，等腰三角形的性质，勾股定理。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  △CDM的周长 = CD + DM + MC。
    2.  CD是定长，因为D和C是定点。
    3.  要求周长最小，就是求 DM + MC 的最小值。
    4.  M在直线EF上，而EF是AC的垂直平分线，根据其性质，MC = MA。
    5.  因此，问题转化为求 DM + MA 的最小值。
    6.  根据“两点之间线段最短”，当D、M、A三点共线时，DM+MA最小，最小值就是线段AD的长度。
    7.  所以，△CDM周长的最小值 = CD + AD。
    8.  利用已知条件（BC=4, D是中点, 面积=16）求出高AD的长度，并求出CD的长度，两者相加即为答案。

#### **第5题**
*   **题目意思**: 在等边三角形ABC中，在AC边上取两点M, N，使得∠MBN = 30°。三条线段AM, MN, CN的长度分别是m, x, n。
*   **题目问题**: 判断以m, n, x为三边构成的三角形的形状。
*   **题目关键字及关键字提供的多个信息**:
    *   **等边三角形ABC**: 三边相等，三个角都是60°。
    *   **∠MBN = 30°**: 一个关键的角度条件。
    *   **AM=m, MN=x, CN=n**: 定义了三条线段。
*   **题目知识点**: 等边三角形的性质，全等三角形的构造与判定，旋转变换，勾股定理逆定理。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  问题是判断m, n, x三边构成的三角形形状，通常需要找到m, n, x之间的数量关系，特别是是否满足m²+n²=x²（或类似的不等式）。
    2.  这是一个典型的旋转构造题。将△ABM绕点B顺时针旋转60°得到△CBF。
    3.  由旋转性质可知：BF=BM, CF=AM=m, ∠FBC=∠ABM, ∠BCF=∠BAM=60°。
    4.  因为∠ABC=60°，∠MBN=30°，所以∠ABM + ∠NBC = 30°。
    5.  又因为∠FBC=∠ABM，所以∠FBC + ∠NBC = 30°，即∠FBN=30°。
    6.  因此∠MBN = ∠FBN = 30°。
    7.  在△MBN和△FBN中，利用SAS证明它们全等。
    8.  由全等可知 FN = MN = x。
    9.  考虑点F, C, N。因为∠BCF=60°，∠BCA=60°，所以F,C,N三点的位置关系需要判断。实际上旋转后F点在三角形外部。考虑∠FCN = ∠BCA + ∠BCF = 60°+60° = 120°（或者在一条直线上，取决于M的位置）。
    10. 在△FCN中，三边为CF=m, CN=n, FN=x。应用余弦定理（或构造直角三角形），可以发现m,n,x之间的关系，从而判断三角形形状。在此特定角度下（120°），会得到一个钝角三角形的关系。

#### **第6题**
*   **题目意思**: 在△ABC中，D是AB中点，DE⊥AB。有一个奇特的条件∠ACE+∠BCE=180°，这意味着A, C, B三点在一条直线上吗？哦，看图，应该是∠ACD+∠BCE=180°的笔误，或者是指外角。题目的表述`∠ACE+∠BCE=180°`意味着A, C, B共线，这与△ABC矛盾。应该是印刷错误，常见的情况是`∠ACD = ∠BCE`或`∠CAD = ∠CBE`之类的。根据图7的形状，最有可能的条件是 E 在 BC 的延长线上，且 AC=AE。让我们假设题目条件是**AC=EC**，并且`DE⊥AB`和`D是AB中点`意味着CE=AE。
    *   **重新审题**: `∠ACE+∠BCE=180°` 意味着点A, C, B在一条直线上，这显然与△ABC矛盾。如果E点在三角形内部，这也不可能。看图6，E在外部，C在AE和BE之间，这条件意味着A,C,B三点共线。看图7，点E在△ABC外部。`∠ACE+∠BCE=180°`意味着A,C,B共线。**这道题的条件可能有误。**
    *   **最可能的修正**: 题目可能是指∠AEC + ∠BEC = 180° (A,E,B共线，不可能) 或者 ∠AC**D** + ∠BC**E**=180° (无此图) 或者 E, C, F三点共线。
    *   **基于图7的合理猜测**: 可能是DE是AB的垂直平分线。所以AE=BE。又因为EF⊥AC，AC=9, BC=7。求AF。这条件也不足。
    *   **让我们尝试理解原始条件 `∠ACE+∠BCE=180°`**: 这意味着A, C, B三点在一条直线上。这不可能构成三角形。
    *   **另一种解释**: C点在直线AB上。这也不对。
    *   **再一种解释**: 可能是 `∠ACF + ∠BCF = 180°` -> A,C,B共线。
    *   **结论**: 此题条件描述不清或有误，无法进行有效分析。假设条件是`DE`为`AB`的垂直平分线，则`AE=BE`。又有`EF⊥AC`。我们已知`AC=9`, `BC=7`。求`AF`。在`Rt△AEF`和`Rt△BEF`中，`EF`是公共边。这依然缺少条件。
    *   **如果条件是 ∠AEC = ∠B**，结合 D 是 AB 中点且 DE⊥AB，则 AE=BE。那么 ∠AEC = ∠B，∠EAB = ∠B。所以 △ABE 和 △AEC 可能是等腰。
    *   **如果按照第7题的图来看**: D是AB中点，DE⊥AB，所以AE=BE。EF⊥AC。已知AC=9, BC=7。求AF。在Rt△AEF中，AF = AE * cos(∠EAF)。在△ABC中，由余弦定理... 这超纲了。
    *   **换个思路**: 也许C, E, F三点共线。`DE⊥AB`, D是AB中点，所以DE是AB的垂直平分线，`AE=BE`。`EF⊥AC`。求`AF`。连接`CE`。在`Rt△AEF`中，`AE² = AF² + EF²`。在`Rt△CEF`中，`CE² = CF² + EF²`。 `AE² - CE² = AF² - CF² = (AF-CF)(AF+CF) = (AF-CF)*AC`。即 `BE² - CE² = 9(AF-CF)`。在`△BCE`中，由勾股定理（如果∠BCE=90）或余弦定理。
    *   **最终判定**: 题目条件`∠ACE+∠BCE=180°`有严重问题，无法解答。如果强行猜测，可能是想表达`AE=BE` (由DE是AB中垂线得到)，然后利用勾股定理 `AE^2-AF^2 = CE^2-CF^2`，即 `BE^2-CE^2 = (AC-CF)^2-CF^2`。这需要知道`BE`和`CE`的关系，题目未给。因此本题跳过。

### **第二部分：填空题**

#### **第7题**
*   **题目意思**: BD是中线，E是BD的中点，F是CE的中点。已知△AEF的面积。
*   **题目问题**: 求△ABC的面积。
*   **题目关键字及关键字提供的多个信息**:
    *   **中线BD**: D是AC的中点，所以S(△ABD) = S(△CBD) = 1/2 * S(△ABC)。
    *   **E是BD的中点**: S(△ABE) = S(△ADE) = 1/2 * S(△ABD)。
    *   **F是CE的中点**: S(△AEF) = S(△ACF) = 1/2 * S(△AEC)。
*   **题目知识点**: 三角形中线的性质（中线平分面积），同底（或等底）等高三角形面积关系。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  这是一个利用中线平分面积性质反向推导的问题。
    2.  F是CE的中点，所以 S(△AEF) = S(△ACF)，因此 S(△AEC) = 2 * S(△AEF) = 2 * 3 = 6。
    3.  E是BD的中点，连接AE, CE。S(△ABE) = S(△ADE)，S(△CBE) = S(△CDE)。
    4.  S(△AEC) = S(△ADE) + S(△CDE)。
    5.  S(△ABC) = S(△ABE) + S(△CBE) + S(△AEC) = S(△ADE) + S(△CDE) + S(△AEC) = 2 * S(△AEC) = 12。
    6.  或者用更系统的方法：设S(△ABC)=S。BD是中线 -> S(△ABD)=S/2。E是BD中点 -> S(△ABE)=S/4。F是CE中点，连接BF。S(△AEF) = 1/2 * S(△AEC)。 S(△AEC) = S(△ABC) - S(△ABE) - S(△BCE) ... 这个方法比较复杂。
    7.  最简单的方法是连接DE。F是CE中点 -> S(△DEF) = S(△DCF)。 S(△AEF) = 3。
    8.  用“燕尾定理”或逐级面积关系推导。从S(△AEF)=3出发，F是CE中点，所以S(△AEC) = 2 * S(△AEF) = 6。E是BD中点，所以 S(△AEC) = S(△ADE) + S(△CDE) = 1/2 S(△ABD) + 1/2 S(△CBD) = 1/2 * (S(△ABD) + S(△CBD)) = 1/2 * S(△ABC)。所以 S(△ABC) = 2 * S(△AEC) = 12。

#### **第8题**
*   **题目意思**: AC和DF交于G，AC=DF。D,C在BE上。有三个角相等：∠B=∠E=∠1 (∠1是∠AGD或∠FGC)。已知BE, AB, EF的长度。
*   **题目问题**: 求CD的长度。
*   **题目关键字及关键字提供的多个信息**:
    *   **AC=DF**: 两条关键线段相等。
    *   **∠B=∠E=∠1**: 多个角度相等，暗示有全等三角形或等腰三角形。∠1通常指对顶角中的一个，如∠AGD。
    *   **D, C在BE上**: 点的共线关系。
*   **题目知识点**: 全等三角形的判定(AAS)和性质，线段的和差。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  观察△ABС和△DFE。已知∠B=∠E。若能证明另外一组角和一组边相等，就可以证全等。
    2.  ∠1 = ∠AGD，其对顶角∠FGC。在△ABС中，∠ACB = 180 - ∠B - ∠BAC。在△GFC中，∠GFC = 180 - ∠FGC - ∠GCF。
    3.  在△ABG中，∠AGB = 180 - ∠B - ∠BAG。
    4.  由∠B=∠E, ∠1=∠AGD=∠FGC, 考虑证明△ABC ≅ △DFE。已知∠B=∠E, AC=DF。还缺一个条件。
    5.  由∠1=∠B，考虑△ABG。∠AGB=180-∠B-∠BAG。∠1=∠B，在△ABG中，这不一定成立。
    6.  由∠E=∠1=∠AGD，在△GED中，∠GDE + ∠GED + ∠DGE = 180。
    7.  换个角度，在△ABС 和 △DFE 中，有∠B=∠E, AC=DF。若能证明∠BCA = ∠EFD，则可用AAS证明全等。
    8.  ∠1 = ∠AGD，所以其补角∠AGF=∠DGC=180-∠1。
    9.  由∠B=∠1(∠AGD)，且它们是△ABG的两个角，这不太可能。应该是∠1指的是图中的∠ACB。**假设∠1=∠ACB**。则∠B=∠E=∠ACB。
    10. 如果∠1=∠ACB，则△ABC是等腰三角形，AB=AC。
    11. 如果∠1是图中标示的∠DGC，那么由外角定理，∠1 = ∠GBC + ∠GCB = ∠B + ∠GCB。而已知∠1=∠B，则∠GCB=0，不可能。
    12. **重新审题，图上∠1标在∠DFE处**。那么条件是 ∠B = ∠E = ∠DFE。这导致E和F重合，或者DF平行于BC，不可能。
    13. **最可能的解释是，图上数字1旁边的弧线指的是∠DGC**。所以∠B=∠E=∠DGC。在△DGC和△FGE中，∠DGC=∠FGE（对顶角）。所以∠E=∠FGE，△FGE是等腰三角形，FG=FE=n。同理，∠B=∠DGC，在△BGC中无法推导。
    14. **让我们假设∠1是∠ACB**。即∠B = ∠E = ∠ACB。则△ABC中，AB=AC=m。在△ABC和△DFE中，∠B=∠E, ∠ACB=??, AC=DF=m。无法证明全等。
    15. **最终的合理猜测：图中的1指的是∠BCA**。所以∠B = ∠E, 且△ABC是等腰三角形（AB=AC）。但题目是求CD。
    16. **尝试证明△ABC ≅ △DFE**。我们有∠B=∠E，AC=DF。还需要一个条件。由∠B=∠E=∠1，如果∠1=∠DFE，则△DFE是等腰三角形。如果∠1=∠BCA，则△ABC是等腰三角形。这似乎是解题的关键。
    17. 假设可以证明 **△ABC ≅ △DFE**。那么 BC=EF=n, AB=DF。但已知AB=m, AC=DF。
    18. 让我们用AAS。在△ABC和△DFE中，∠B=∠E，∠BCA=∠EFD(需要证明)，AC=DF(已知)。如果能证角度相等，则△ABC≅△DFE。则BC=EF=n, AB=DE。
    19. 一个经典模型：若∠B=∠E, ∠ACB=∠DFE, AB=DF，则△ABC≅△DFE。这里是AC=DF。
    20. 重新看条件，可能是△ABG和△FEG相似或全等。
    21. **最直接的思路**: 证明△ABC ≅ △DEF。在△ABC与△DEF中，有∠B=∠E，AC=DF，若能加上∠A=∠D，则AAS全等。由BE=1, AB=m, EF=n，CD = BC-BD 或 BE-BC-DE。
    22. **正确思路**: 在△ABC和△DFE中，∠B=∠E，AC=DF，再找一个角。因为∠1是对顶角，∠AGD=∠FGC。在△ABG中，∠A = 180 - ∠B - ∠AGB。在△DFG中，∠D=180-∠DFG-∠DGF。
    23. 考虑△ABC和△EDF。∠B=∠E, AC=DF。若 BC=EF=n，则可以用SSA，但不一定全等。
    24. 答案是m-n。这暗示了CD = AB - EF。这通常来自于全等后的线段加减。即 BC=DE。CD = BE - BC - DE = BE - 2BC = 1-2n? 不对。CD = BC-BD。
    25. 如果 **BC = DE**。CD = CE - DE = CE - BC。BE = BC+CE = 1。CD = (BC+CD) - DE = BC - DE + CD。
    26. 假设 **△ABC ≅ △EDF** (注意是EDF不是DFE)。则∠B=∠E, BC=DF, AC=EF=n。这与AC=DF矛盾。
    27. 假设 **△GBC ≅ △GED**。
    28. **正确解法**: 证明 **△ABC ≅ △FDE**。 在BE上取点H，使CH//DF。这太复杂了。
    29. 让我们假设结论 **CD = m-n**。这通常意味着 **BC+CD = m** 且 **BC=n**？或者 **BD = m**, **BC=n**？
    30. 应该是 **△ABC ≅ △DEF**。那么 BC=EF=n。AB=DE=m。BE=BD+DE=BD+m=1? 不对。BE=BC+CE=n+CE=1。 CD在BC和CE之间。CD = BC+DE-BE = n+m-1?
    31. 让我们假设**△GAC ≅ △GDF**。AC=DF，∠AGC=∠DGF，还需要一个角。
    32. **最终思路**: 过点C作CP//DF交AF于P，过点A作AP//DF交DF延长线于P。
    33. 这道题条件可能不完整或存在特殊模型。一个可能的模型是△ABC ≅ △GEF。
    34. 答案 `m-n` 提示性很强。尝试证明 **BC+EF = AB**。 即 BC+n=m。
    35. 可能是通过构造全等，把AB和EF转移到一条线上与BC比较。
    36. 设法证明 **BC=DE**。CD = BE - BC - DE = BE - 2BC。
    37. 正确思路是证明 **△ABC ≅ △DEF**。由∠B=∠E, AC=DF, 还需一个角相等。若∠BCA=∠EFD，则AAS成立。则BC=EF=n，AB=DE=m。CD = BE - BC - DE = 1 - n - m。这肯定不对。

    39. 最终猜测：通过全等证明了 **BC=EF=n** 并且 **CE=AB-EF**。

#### **第9题**
*   **题目意思**: 在一个直角三角形ABC中，三边长已知。O是两个锐角（∠CAB和∠CBA）的角平分线的交点。
*   **题目问题**: 求点O到斜边AB的距离。
*   **题目关键字及关键字提供的多个信息**:
    *   **Rt△ABC, ∠C=90°**: 直角三角形。
    *   **AC=4, BC=3, AB=5**: 勾股数，三边确定。
    *   **O是角平分线的交点**: O是△ABC的内心。
    *   **点O到AB的距离**: 内心到三边的距离相等，这个距离就是内切圆的半径r。
*   **题目知识点**: 角平分线的性质，三角形内心的性质，三角形面积公式。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  识别出点O是三角形的内心。
    2.  利用内心性质：内心到三边的距离相等，都等于三角形内切圆的半径r。
    3.  问题转化为求内切圆半径r。
    4.  利用面积法求解r。S(△ABC) = S(△OAB) + S(△OBC) + S(△OAC)。
    5.  S(△ABC) = 1/2 * AC * BC = 1/2 * 4 * 3 = 6。

    7.  建立方程 6 = 1/2 * r * (5+3+4)，解出r即可。

#### **第10题**
*   **题目意思**: 在一个顶角为40°的等腰三角形ABC (AB=AC)中，点D是腰AB上的一个动点。连接CD，并将三角形ADC沿着CD翻折，得到一个新的三角形A'DC。翻折后，边A'C与边AB相交于点E。已知△A'DE是一个等腰三角形。
*   **题目问题**: 求角∠ACD的度数。
*   **题目关键字及关键字提供的多个信息**:
    *   **△ABC中, AB=AC, ∠A=40°**: 这是一个确定的等腰三角形，可以算出其底角∠B = ∠ACB = (180°-40°)/2 = 70°。
    *   **翻折得到△A'DC**: 翻折是一种轴对称变换，它保证了△ADC ≅ △A'DC。由此可得一系列相等关系：
        1.  边相等: AD=A'D, AC=A'C。
        2.  角相等: ∠ACD=∠A'CD, ∠CAD=∠CA'D=40°, ∠ADC=∠A'DC。
    *   **△A'DE是等腰三角形**: 这是解题的核心条件。它意味着△A'DE的三条边中，有两条是相等的。这需要进行分类讨论，共有三种可能情况：
        1.  两腰为A'D, A'E (A'D = A'E)。
        2.  两腰为DE, DA' (DE = DA')。
        3.  两腰为ED, EA' (ED = EA')。
*   **题目知识点**: 等腰三角形的性质（等边对等角，等角对等边），轴对称（翻折）的性质，三角形内角和定理，三角形外角定理，分类讨论思想。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  设待求的角 ∠ACD = x。根据翻折性质，∠A'CD = x，所以∠ACA' = 2x。
    2.  利用“△A'DE是等腰三角形”这一核心条件，分三种情况进行讨论，目标是建立一个关于x的方程。
    3.  **情况一: DE = DA'**。
        *   由翻折可知 DA' = DA，所以 DE = DA。
        *   在△A'DE中，边DE和DA'所对的角分别是∠DA'E和∠A'ED。所以∠DA'E = ∠A'ED。
        *   由翻折可知 ∠DA'E = ∠A = 40°，因此 ∠A'ED = 40°。
        *   ∠A'ED与∠AEC是同一个角，所以∠AEC = 40°。
        *   在△AEC中，我们已知∠A=40°, ∠AEC=40°，所以它是一个等腰三角形，且∠ACE = 180° - 40° - 40° = 100°。
        *   但是，E点在边AB上，所以∠ACE必然是∠ACB的一部分（或全部），因此∠ACE ≤ ∠ACB = 70°。100° > 70°，产生矛盾。所以此情况不可能。
    4.  **情况二: ED = EA'**。
        *   在△A'DE中，边ED和EA'所对的角分别是∠EA'D和∠EDA'。所以∠EA'D = ∠EDA'。
        *   已知 ∠EA'D = ∠A = 40°，所以 ∠A'DE = 40°。
        *   此时，在△A'DE中，∠A'ED = 180° - 40° - 40° = 100°。
        *   所以 ∠AEC = 100°。在△AEC中，∠A=40°，可求得∠ACE = 180° - 100° - 40° = 40°。
        *   现在我们有了两个关于角的等式：一个是从几何位置得到的∠ACE，另一个是通过翻折关系得到的。
        *   在△A'EC中，∠EA'C=40°, ∠A'EC=100°, 所以∠A'CE = 180° - 40° - 100° = 40°。
        *   根据图形，∠ACA' = ∠ACE + ∠A'CE (或 |∠ACE - ∠A'CE|，取决于A'位置)。从合理的图形位置看，应为相加。
        *   我们有 ∠ACA' = 2x，所以 2x = ∠ACE + ∠A'CE = 40° + 40° = 80°。
        *   解得 x = 40°。该解没有矛盾，是一个可能的答案。
    5.  **情况三: A'D = A'E**。
        *   在△A'DE中，∠A'DE = ∠A'ED = (180° - 40°)/2 = 70°。
        *   所以 ∠AEC = 70°。在△AEC中，∠A=40°，可求得∠ACE = 180° - 70° - 40° = 70°。
        *   ∠ACE = 70°，而∠ACB = 70°。这意味着射线CE与射线CB重合。
        *   因为E在直线AB上，C在直线CB上，它们的交点就是B点。所以 E=B。
        *   当E=B时，A'C与AB的交点是B，意味着A', C, B三点共线。
        *   在△A'BC中，由翻折知A'C=AC，又AC=AB，所以A'C=AB。此时∠A'BC = ∠B = 70°，∠CA'B = ∠A = 40°。所以∠A'CB = 180°-70°-40°=70°。
        *   ∠A'CB=70°≠180°，所以A', C, B不共线。此情况产生矛盾。
    6.  综上所述，只有一种情况成立，解得∠ACD = 40°。

#### **第11题**
*   **题目意思**: 已知三条线段AC, AB, BD的长度相等，角ABD是一个直角，并且线段BC的长度是8。
*   **题目问题**: 求三角形BCD的面积。
*   **题目关键字及关键字提供的多个信息**:
    *   **AC=AB=BD**: 提供了等腰三角形△ABC，以及一个与外部线段的相等关系。
    *   **∠ABD=90°**: 提供了直角，这是使用勾股定理或建立坐标系的关键。
    *   **BC=8**: 提供了一个具体的长度值。
*   **题目知识点**: 三角形面积公式，坐标系的建立与应用，两点间距离公式。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  要求△BCD的面积，根据面积公式 S = (1/2) * 底 * 高。我们可以视BD为底，那么高就是点C到直线BD的距离。
    2.  利用“直角”这个关键字，建立直角坐标系会使问题非常清晰。
    3.  设B为坐标原点(0,0)。因为∠ABD=90°，可以将BD放在x轴上，AB放在y轴上。
    4.  设 AC=AB=BD=x (x>0)。则各点坐标为：B(0,0), D(x,0), A(0,x)。
    5.  点C的坐标是未知的，设为 C(m, n)。
    6.  利用已知的两个距离条件来确定点C的坐标（或其关键分量）。
        *   条件一: C到B的距离是8。根据两点距离公式，m² + n² = 8² = 64。
        *   条件二: C到A的距离是x。根据两点距离公式，(m-0)² + (n-x)² = x²，即 m² + n² - 2nx + x² = x²。
    7.  简化条件二的方程，得到 m² + n² - 2nx = 0。
    8.  将条件一 (m² + n² = 64) 代入简化后的条件二，得到 64 - 2nx = 0。
    9.  从 64 - 2nx = 0 解出 n = 64 / (2x) = 32/x。
    10. 这里的n就是点C的纵坐标，它代表的正是点C到x轴（即直线BD）的距离，也就是△BCD的高。
    11. 现在可以计算△BCD的面积了：S(△BCD) = (1/2) * 底(BD) * 高(n) = (1/2) * x * (32/x)。
    12. 在计算中，未知数x被消掉了，直接得到面积 S = (1/2) * 32 = 16。

#### **第13题**
*   **题目意思**: 已知一个三角形ABC (其中AC > AB)，和一条长度等于BC的线段DE。要求以D, E为顶点，作出与△ABC全等的三角形。

    *   **DE=BC**: 指明了新三角形的一条边DE已经对应于△ABC的边BC。
*   **题目知识点**: 全等三角形的判定（特别是SSS），尺规作图的基本原理（两弧交点确定第三个顶点）。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  设要作的三角形是△PQR，它与△ABC全等。已知其中两个顶点是D, E，且DE=BC。那么第三个顶点，我们称之为P。
    2.  新三角形是△PDE。它的三边是PD, PE, DE。
    3.  根据全等条件，△PDE的三边必须与△ABC的三边 {AB, BC, AC} 对应。我们已经有 DE=BC。

        *   **情况一**: PD = AB 并且 PE = AC。
        *   **情况二**: PD = AC 并且 PE = AB。
    5.  分析每一种情况能作出多少个三角形：
        *   对于**情况一 (PD=AB, PE=AC)**:
            *   点P必须同时满足两个条件：① 到点D的距离为AB，② 到点E的距离为AC。

            *   同理，点P是以D为圆心、AC为半径的圆和以E为圆心、AB为半径的圆的交点。

#### **第14题**
*   **题目意思**: 在一个Rt△ABC(∠C=90°, ∠B=30°, BC=9)中，点P在直角边BC上移动，点D在斜边AB上移动。
*   **题目问题**: 求AP+PD的最小值。
*   **题目关键字及关键字提供的多个信息**:
    *   **Rt△ABC, ∠B=30°, BC=9**: 这是一个确定的30-60-90直角三角形。
    *   **P为边BC上的动点**: P的运动轨迹是一条线段。
    *   **D为边AB上的动点**: D的运动轨迹也是一条线段。
    *   **AP+PD的最小值**: 连接一个定点(A)和两个动点(P,D)的路径和的最小值问题。
*   **题目知识点**: 轴对称（反射）求最短路径，点到直线的距离。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  问题是求 A→P→D 的路径总长最小值，其中P在直线BC上，D在直线AB上。
    2.  这是一个涉及两次反射的“台球”或“光路”模型。
    3.  第一步：解决动点P。为了使AP+PD最短（暂时固定D），我们需要对点A或点D做关于直线BC的对称。我们选择对定点A作关于直线BC的对称，得到对称点A'。
        *   根据轴对称性质，AP = A'P。
        *   因此，AP+PD = A'P+PD。
    4.  第二步：连接A'和D。当P点位于线段A'D与直线BC的交点时，A'P+PD取得最小值，这个最小值就是线段A'D的长度。
    5.  现在问题转化为：求A'D的最小值，其中D是AB上的动点。
    6.  点A'是定点（因为A和直线BC都是固定的），动点D的轨迹是定直线AB。
    7.  “求定点A'到动点D(在直线AB上)的距离的最小值”，这个最小值就是定点A'到直线AB的垂直距离。
    8.  所以，解题的核心是计算点A'到直线AB的距离。
    9.  计算过程：
        *   确定A'的位置。A'是A关于BC的对称点。因为∠ACB=90°，所以A'在直线AC的延长线上，并且A'C = AC。
        *   在Rt△ABC中，BC=9, ∠B=30°，可求出 AC = BC*tan(30°) = 3√3, AB = BC/cos(30°) = 6√3。
        *   所以A'C = 3√3。
        *   考虑△A'AB。我们知道边AB的长度，需要找到A'到AB的高。
        *   在△A'BC中，A'C=AC, BC是公共边, ∠A'CB=∠ACB=90°，所以△A'BC ≅ △ABC。
        *   因此，A'B = AB，∠A'BC = ∠ABC = 30°。
        *   在△A'BA中，A'B=AB，∠A'BA = ∠A'BC + ∠ABC = 30° + 30° = 60°。
        *   一个顶角为60°的等腰三角形是等边三角形。所以△A'BA是等边三角形。
        *   A'到直线AB的距离就是等边△A'BA的高。
        *   高 = (√3/2) * 边长 = (√3/2) * AB = (√3/2) * 6√3 = 9。

#### **第16题**
*   **题目意思**: 在一个Rt△ABC (AC=3, BC=4)中，顶点C在一条直线L上。有两个动点P和Q，P从A出发沿A→C→B路径运动（速度为1），Q从B出发沿B→C→A路径运动（速度为2），同时开始。在某一时刻t，分别从P, Q向直线L作垂线PE和QF，使得直角三角形PEC和直角三角形CFQ全等。

    *   **PE⊥L, QF⊥L**: 构造了两个直角三角形△PEC和△CFQ。

    4.  所以，解题的关键是建立方程 PC = CQ。
    5.  由于P和Q的运动是分段的，需要根据时间t的范围，分类讨论P和Q的位置，并用t表示出PC和CQ的长度。
    6.  **P的运动**:
        *   当 0 ≤ t ≤ 3, P在AC上，AP=t, PC=3-t。
        *   当 3 < t ≤ 7, P在CB上，P走过的路程是3+CP，即t=3+CP，所以PC=t-3。

        *   当 0 ≤ t ≤ 2 (路程2t≤4), Q在BC上，BQ=2t, CQ=4-2t。

    8.  **分类讨论建立方程**:
        *   **Case 1: 0 ≤ t ≤ 2**。P在AC上(PC=3-t)，Q在BC上(CQ=4-2t)。

        *   **Case 3: 3 < t ≤ 3.5**。P在CB上(PC=t-3)，Q在CA上(CQ=2t-4)。

            方程: t-3 = 3  =>  t=6。 (3.5 < 6 ≤ 7，在此范围内，是第三个解)。
    9.  将所有求得的t值汇总，即为答案。

#### **第21题**
*   **题目意思**: 在△ABC中，已知AB>AC，AD是BC边上的高。
*   **题目问题**: 求证 BD>DC。
*   **题目关键字及关键字提供的多个信息**:
    *   **AB > AC**: 边长的不等关系。
    *   **AD⊥BC**: AD是高，构造了两个直角三角形△ABD和△ACD。
*   **题目知识点**: 勾股定理，不等式的性质。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  要证明两条线段BD和DC的不等关系。
    2.  这两条线段分别位于两个直角三角形△ABD和△ACD中。
    3.  利用“高”这个关键字，可以在这两个直角三角形中使用勾股定理。
    4.  在Rt△ABD中，根据勾股定理有：AB² = AD² + BD²。
    5.  在Rt△ACD中，根据勾股定理有：AC² = AD² + DC²。
    6.  将这两个等式变形，用BD和DC表示出来：
        *   BD² = AB² - AD²
        *   DC² = AC² - AD²
    7.  题目已知条件是 AB > AC。因为边长是正数，所以两边平方后不等关系不变：AB² > AC²。
    8.  现在比较 BD² 和 DC²。用已知的 AB² > AC² 进行代换。
    9.  在不等式 AB² > AC² 的两边同时减去同一个数 AD²，不等号方向不变。
    10. 所以，AB² - AD² > AC² - AD²。
    11. 即 BD² > DC²。
    12. 因为BD和DC都是线段长度，是正数，所以开方后不等关系不变。
    13. 因此，BD > DC。证明完毕。

        *   因为∠A=90°，所以旋转后 A, C, E 在一条直线上，且∠CAE=90°？不一定。
        *   旋转后 ∠EAB=∠CAD。∠EAC = ∠EAB+∠BAC。
        *   这个方法不直接。
    12. **重新审题后最直接的辅助线**: 从D点作一条垂线DF到BC的延长线上。
        *   在四边形ABFD中，∠A=∠B=∠F=90°，所以是矩形。
        *   这个方法需要∠B=90°，题目没给。
    13. **正确解法**: 将△ABD补全成正方形。或者将△ADC "移"出来。
        *   作辅助线：过点D作DE⊥BC交BC延长线于E。
        *   延长AD至F，使DF=BE。
    14. **最终的简洁解法**: 将△DAB绕点A旋转270°（或-90°），使AD与AB重合。不对。
        *   将△ADC向外平移。
        *   **最有效的方法是“割补法”**：将△ADC切下来，补到AB边上。
        *   在BC延长线上取一点E，使得CE = CD。不对。
        *   在AB延长线上取一点E，使AE=AC。
        *   **正确思路**: 将△ADC绕A点旋转，使AD与AB重合。这需要旋转270度。
        *   **让我们尝试将△ABC绕A点旋转**...
        *   **最终解法**: 过D作BC的平行线，过B作CD的平行线。
        *   **正确的辅助线**：过D作DF⊥CB，交CB延长线于F。在Rt△DCF中，DF²+CF²=CD²。
        *   在DF上取一点G，使DG=AB。
        *   **好吧，经典解法是旋转**: 将△ABD绕点A旋转90°，使得AB与AD重合。这也不对。
        *   **正确解法**: 将△AD C 绕点 A 旋转 270°（-90°），得到△ABE，其中D→B，C→E。
        *   那么AE=AC，BE=DC，∠AEB=∠C=90°。
        *   此时∠EAB=∠CAD。因为∠DAB=90°，所以∠CAD+∠CAB=90°。所以∠EAB+∠CAB=90°，即∠EAC=90°。
        *   现在我们有了一个Rt△EAC，和一个点B。
        *   考虑点E, B, C。在Rt△EBC中，EC² = EB² + BC²。
        *   我们有 BE=DC, BC+CD=6 => BC+BE=6。
        *   在Rt△AEC中，由勾股定理AC² = AE²... Wait AE=AC. 所以是等腰直角△AEC。
        *   EC² = AC² + AE² = 2AC²。
        *   所以 2AC² = (DC)² + (BC)²。
        *   这个关系不足以求解。
        *   **有一个更巧妙的思路**: 面积 S(ABCD) = S(△ABD) + S(△BCD)。S(△ABD) = 1/2 * AB²。S(△BCD) = 1/2 * BC * CD * sin(C) = 1/2 BC*CD。
        *   所以 S = 1/2 AB² + 1/2 BC*CD。我们还是不知道AB, BC, CD。
        *   **最终正确方法**: 连接BD。在Rt△ABD中，BD²=2AB²。在Rt△BCD（不是直角），由余弦定理 BD² = BC²+CD² - 2BC*CD*cos(C)。∠C=90°，所以 BD²=BC²+CD²。
        *   所以我们有 2AB² = BC² + CD²。
        *   我们要求面积 S = S(△ABD) + S(△BCD) = 1/2 AB² + 1/2 BC*CD (因为∠C=90°，高就是边)。
        *   用 1/2(BC²+CD²) 替换 AB²。S = 1/4(BC²+CD²) + 1/2 BC*CD = 1/4 (BC²+2BC*CD+CD²) = 1/4 (BC+CD)²。
        *   已知 BC+CD=6。所以 S = 1/4 * (6)² = 9。

#### **第23题**
*   **题目意思**: 这是一个证明题，要求证明一个关于三角形全等的判定定理。
*   **题目问题**: 证明：如果两个三角形的斜边和斜边上的高分别相等，那么这两个三角形全等。
*   **题目关键字及关键字提供的多个信息**:
    *   **斜边**: 暗示了所讨论的三角形是直角三角形。
    *   **斜边上的高**: 从直角顶点向斜边所作的垂线。
    *   **分别相等**: 提供了两组相等的线段作为已知条件。
    *   **三角形全等**: 证明的目标。
*   **题目知识点**: 全等三角形的判定（HL, AAS等），勾股定理，三角形面积公式。
*   **题目问题、题目答案与关键字的联系 (解题思路)**:
    1.  首先，要明确这是一个普适性的证明，需要设定两个任意的三角形来表示。
    2.  设这两个三角形分别为△ABC和△A'B'C'。
    3.  根据题意，这两个三角形都是直角三角形。不妨设∠C = ∠C' = 90°。
    4.  写出已知条件：
        *   斜边相等: AB = A'B'。
        *   斜边上的高相等: 设CD是△ABC斜边AB上的高，C'D'是△A'B'C'斜边A'B'上的高。则 CD = C'D'。
    5.  写出求证目标：△ABC ≅ △A'B'C'。
    6.  要证明两个直角三角形全等，最常用的方法是HL（斜边、直角边）。我们已经有斜边AB = A'B'，如果能证明任意一组直角边相等（如AC=A'C'或BC=B'C'），问题就解决了。
    7.  思路一：利用面积法。
        *   三角形的面积有两种表示方法：S(△ABC) = (1/2) * AC * BC，以及 S(△ABC) = (1/2) * AB * CD。
        *   所以 AC * BC = AB * CD。
        *   同理，A'C' * B'C' = A'B' * C'D'。
        *   因为 AB = A'B' 且 CD = C'D'，所以 AC * BC = A'C' * B'C'。 (①)
    8.  思路二：利用勾股定理。
        *   在△ABC中，AC² + BC² = AB²。
        *   在△A'B'C'中，A'C'² + B'C'² = A'B'²。
        *   因为 AB = A'B'，所以 AC² + BC² = A'C'² + B'C'²。 (②)
    9.  现在我们有两个关于四条直角边 (AC, BC, A'C', B'C') 的方程组：
        *   { AC * BC = A'C' * B'C'  (①)
        *   { AC² + BC² = A'C'² + B'C'²  (②)
    10. 目标是证明 AC = A'C' 且 BC = B'C' (或者 AC = B'C' 且 BC = A'C')。
    11. 对方程组进行代数变换。
        *   观察 (AC + BC)² = AC² + BC² + 2*AC*BC
        *   观察 (A'C' + B'C')² = A'C'² + B'C'² + 2*A'C'*B'C'
        *   由(①)和(②)可知，(AC + BC)² = (A'C' + B'C')²。因为边长为正，所以 AC + BC = A'C' + B'C'。(③)
        *   同理，观察 (AC - BC)² = AC² + BC² - 2*AC*BC
        *   观察 (A'C' - B'C')² = A'C'² + B'C'² - 2*A'C'*B'C'
        *   由(①)和(②)可知，(AC - BC)² = (A'C' - B'C')²。所以 |AC - BC| = |A'C' - B'C'|。(④)
    12. 联立方程组 { AC + BC = A'C' + B'C' (③) 和 |AC - BC| = |A'C' - B'C'| (④) }。
    13. 解这个方程组。
        *   如果 AC - BC = A'C' - B'C'，与③式相加得 2AC = 2A'C' => AC = A'C'。代回③得 BC = B'C'。
        *   如果 AC - BC = -(A'C' - B'C') = B'C' - A'C'，与③式相加得 2AC = 2B'C' => AC = B'C'。代回③得 BC = A'C'。
    14. 无论哪种情况，都证明了直角边对应相等。
    15. 根据HL定理（或者SSS，SAS），△ABC ≅ △A'B'C'。证明完毕。
