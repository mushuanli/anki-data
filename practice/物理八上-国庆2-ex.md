### **一. 选择题**

#### **题号 1**
*   **题目意思:** 分析四个与声音有关的诗句，并判断哪个物理学解释是正确的。
*   **题目问题:** 哪个诗句的声学解释是正确的？
*   **题目关键字:** 乡音无改，笛声，高声，闻啼鸟。
*   **题目知识点:** 声音的三要素（音调、响度、音色），声音的产生（振动），声音的传播（需要介质）。
*   **题目问题、题目答案与关键字的联系:**
    *   A. “乡音无改”指的是口音、音色（Timbre）没有改变，而不是响度（Loudness）。所以A错误。
    *   B. “笛声”是由笛子内部的空气柱振动产生的，而不是笛子本身振动。所以B的描述不准确。
    *   C. “不敢高声语”中的“高”指的是声音大，即响度大，而不是音调（Pitch）高。所以C错误。
    *   D. “闻啼鸟”意味着人能听到鸟叫声，声音从鸟的声带发出，通过“空气”这种介质“传入人耳”。这个描述是正确的。
    *   **答案: D**

#### **题号 2**
*   **题目意思:** 琴和瑟是弦乐器，通过弹拨琴弦发声。分析关于它们发声的几个说法哪个正确。
*   **题目问题:** 关于琴和瑟发声的说法，哪个是正确的？
*   **题目关键字:** 琴，瑟，弹拨琴弦，发声。
*   **题目知识点:** 声音的产生（振动），声音的三要素（音调、响度、音色），影响音调的因素（弦的长度、粗细、松紧）。
*   **题目问题、题目答案与关键字的联系:**
    *   A. 弹琴时，声音是由“琴弦”的振动产生的，而不是手指的振动。所以A错误。
    *   B. 琴和瑟发出的声音可以用音调、响度、音色来区分，响度只是其中一个特性。这个说法本身没错，但不全面且不是最佳选项。
    *   C. 用力越大，琴弦的振幅越大，发出声音的响度越大，而不是音调越高。所以C错误。
    *   D. 对于“同一根琴弦”，振动部分“越长”，振动频率越低，发出的声音“音调就越低”。这是影响音调高低的基本规律。所以D正确。
    *   **答案: D**

#### **题号 6**
*   **题目意思:** 小华站在平面镜前2m处，分析关于平面镜成像的几个说法，找出不正确的。
*   **题目问题:** 关于平面镜成像的说法，哪个不正确？
*   **题目关键字:** 平面镜前2m，手指与像相距6mm，移动速度1m/s，照镜子，镜子打碎。
*   **题目知识点:** 平面镜成像的特点（等大、等距、垂直、虚像）。
*   **题目问题、题目答案与关键字的联系:**
    *   A. 手指与像相距6mm，根据像到镜面的距离等于物到镜面的距离，手指到镜面距离为3mm，像到镜面距离也为3mm。所以镜面厚度为3mm，而不是6mm。所以A的结论错误。
    *   B. 原来人距镜2m，像也距镜2m。人以1m/s移动1s，向前移动了1m，现在人距镜1m，像也距镜1m。此时人与像的距离是1m+1m=2m。说法正确。
    *   C. 照镜子时，人是物体，人越亮，像越清晰。应该用手电筒照亮人脸，而不是照镜中的像，因为像是虚像，不能被照亮。所以C的做法错误。
    *   D. 平面镜成像的大小只与物体大小有关，与镜子大小无关。即使镜子碎成两半，每一半仍然可以成一个“完整的像”，只是可能视野变小。说法正确。
    *   题目要求选不正确的。A和C都不正确。但A的计算错误更直接，而C是生活常识应用错误。通常此类问题中C的错误更典型。让我们重新审视A。手指与像相距6mm，则手指到像的距离是6mm。镜面有厚度，光线在前后两个表面反射。如果考虑玻璃的厚度d，物距为x，像距为x+2d(n-1)/n，情况复杂。但中学阶段通常简化为：手指到前表面距离s1, 像到后表面距离s2，s1+s2+d=6mm。这无法直接解出d=6mm。所以A的推论是错的。C的说法也是绝对错误的。在单选题中，C的错误是关于如何使像更清晰的原理性错误，更为常见。
    *   **答案: C** (此题可能存在争议，A的推论也错误，但C是更常见的考点和错误认知)。

#### **题号 9**
*   **题目意思:** 观察者在d点通过水池bc看电线杆aa'在水中的像，求能看到的像的长度。
*   **题目问题:** 观察者能看到电线杆在水中所成像的长度是多少？
*   **题目关键字:** 水池，电线杆，像，观察者，高度尺寸。
*   **题目知识点:** 平面镜成像，光的反射定律，相似三角形。
*   **题目问题、题目答案与关键字的联系:** 水面bc相当于平面镜。观察者的视线范围决定了能看到像的部分。光路是可逆的，从观察者眼睛发出的光线，经过水池边缘（b和c）反射后，所能照到的电线杆高度范围，就是能看到的像的范围。
    *   设观察者眼睛位置为E，地面上d点正上方1.5m处。
    *   光线Ec反射后到达电线杆的最高点P1：利用相似三角形。△(E到地面的垂线与水平线的交点, d, c) ~ △(P1, a, c)。(1.5m / cd) = (P1的高度 / ac)。cd=2m，ac=ab+bc=4+2=6m。所以 1.5/2 = P1的高度/6，解得P1的高度 = 4.5m。
    *   光线Eb反射后到达电线杆的最低点P2：利用相似三角形。△(E到地面的垂线与水平线的交点, d, b) ~ △(P2, a, b)。(1.5m / bd) = (P2的高度 / ab)。bd=bc+cd=2+2=4m，ab=4m。所以 1.5/4 = P2的高度/4，解得P2的高度 = 1.5m。
    *   能看到的电线杆长度为 P1的高度 - P2的高度 = 4.5m - 1.5m = 3m。因为像是等大的，所以能看到的像的长度也是3m。
    *   **答案: C**

#### **题号 10**
*   **题目意思:** 眼睛在(3,0)，平面镜在y=3处，端点为(-2,3)和(0,3)。光源S从原点沿x轴负方向运动，问在哪个区间内能看到S的像，以及像的运动情况。
*   **题目问题:** 光源在哪个区域运动时人眼能看到像？像如何运动？
*   **题目关键字:** 坐标，平面镜，像，运动。
*   **题目知识点:** 平面镜成像特点（对称性），光的反射，可见范围的确定。
*   **题目问题、题目答案与关键字的联系:**
    1.  **找像的位置和运动：** 平面镜在y=3。光源S在(x, 0)上运动。根据对称性，S的像S'的坐标为(x, 6)。当S沿x轴负方向匀速运动时，其像S'也沿平行于x轴的直线y=6向负方向匀速运动。
    2.  **找可见范围：** 人眼E(3,0)能看到像S'，意味着S'发出的光线（实际上是S发出的光经镜面反射的光线）能进入眼睛。可见范围由镜子的两端点决定。连接眼睛E和镜子左端点M1(-2,3)并延长，与像所在的直线y=6相交于A点。连接眼睛E和镜子右端点M2(0,3)并延长，与y=6相交于B点。像S'只有在AB之间时才能被看到。
    3.  **计算A, B坐标：**
        *   直线EM1的方程：利用两点式。斜率 k1 = (3-0)/(-2-3) = -3/5。方程 y-0 = -3/5(x-3)。令y=6，则 6 = -3/5(x-3) => -10 = x-3 => x = -7。所以A点坐标是(-7, 6)。
        *   直线EM2的方程：斜率 k2 = (3-0)/(0-3) = -1。方程 y-0 = -1(x-3)。令y=6，则 6 = -x+3 => x = -3。所以B点坐标是(-3, 6)。
    4.  **结论：** 像S'必须在x坐标为-7到-3的区间内才能被看到。由于像的x坐标与物体的x坐标相同，所以光源S必须在x坐标为-7到-3的区间内运动时，其像才能被人眼看到。
    5.  **分析选项：**
        *   可见区域是 -3 ~ -7 区间。
        *   像S'在y=6的直线上，与S同步沿x轴负方向匀速运动。
        *   B选项：-3 ~ -7 区间，沿x轴负方向匀速运动。这与我们的分析一致。
    *   **答案: B**

#### **题号 11**
*   **题目意思:** 一条平行于ON的光线AB，在两个夹角为θ的平面镜间反射30次后，沿原路返回。求夹角θ。
*   **题目问题:** 两平面镜的夹角θ是多少？
*   **题目关键字:** 平面镜，夹角θ，平行，30次反射，原路返回。
*   **题目知识点:** 光的反射定律，多次反射光路分析。
*   **题目问题、题目答案与关键字的联系:**
    *   “原路返回”意味着最后一次反射，光线是垂直入射到镜面上的。
    *   光线AB平行于ON，第一次入射到OM上。设入射角为α₁。因为AB//ON，所以α₁ = θ。
    *   根据反射定律，反射角也为θ。反射光线与OM的夹角为90°-θ。
    *   此反射光线射到ON上，入射角为α₂ = 90° - (90°-θ) - θ = θ (几何关系)。不对，应该是入射光线与镜面的夹角是θ。那么入射角是90-θ。
    *   让我们重新分析：入射光线与ON平行，则它与OM的夹角为θ。这是入射光线与镜面的夹角。所以第一次入射角为 i₁ = 90° - θ。
    *   经过n次反射后光线方向改变的角度为 2nθ 或 n*180 - 2θ 等，比较复杂。
    *   有一个结论：光线在两个平面镜之间反射，出射光线与入射光线的夹角 δ = 2θ。
    *   更简单的规律：对于互相垂直的镜面，反射两次后光线反向平行。对于任意夹角θ，光线每反射一次，相对于前一镜面的入射方向偏转了2倍的入射角，或者说光线方向每次偏转的角度与镜面夹角有关。
    *   一个重要结论：光线经两平面镜反射n次后，总偏向角为2nθ。但这不适用于本题。
    *   让我们回到“原路返回”的条件：最后一次反射垂直于镜面。
    *   设在OM上反射k次，在ON上反射m次，总共k+m=30次。
    *   最后一次反射（第30次）垂直于镜面。假设它发生在OM上。
    *   光线每次从一个镜面反射到另一个镜面，入射角的变化遵循一个规律。
    *   第一次入射到OM，入射角i₁=90°-θ。反射后到ON，入射角i₂=90°-2θ。再反射到OM，入射角i₃=90°-3θ...第n次反射的入射角为 |90°-nθ|。
    *   “原路返回”意味着最后一次（第30次）在某个镜面上是垂直入射，即入射角为0。
    *   所以 90° - 30θ = 0。解得 θ = 3°。
    *   **答案: A**

#### **题号 12**
*   **题目意思:** 巡逻车测速仪在2.7秒内先后发出两个超声波脉冲P1, P2，并接收到反射信号n1, n2。根据图(b)中的时间信息和声速，判断哪个说法正确。
*   **题目问题:** 根据图示和数据，哪个说法是正确的？
*   **题目关键字:** 超声波，测速，时间差，匀速行驶。
*   **题目知识点:** 回声测距，速度公式 v = s/t。
*   **题目问题、题目答案与关键字的联系:**
    *   从图(b)可以看出：
        *   第一次发出到接收的时间间隔 t₁ = 0.9s - 0.45s = 0.45s。 (选项A说间隔时间是0.45s，这是对的。)
        *   第一次测速时，汽车距离巡逻车的距离 s₁ = v_声 * t₁ / 2 = 340 * 0.45 / 2 = 76.5 m。
        *   第二次发出信号是在 t=2.7s + 0.45s = 3.15s。

*   **题目问题、题目答案与关键字的联系:**
    *   (1) 选用玻璃板代替平面镜，是因为玻璃板透明，既能反射成像，又能看到后面的物体，这样做的目的是 **便于确定像的位置**。
    *   (2) 平面镜成像是物体上每一点发出的光线经镜面反射形成的。遮挡一半镜面，只是减少了参与成像的光线数量，但物体上每一点仍然能通过剩下的一半镜面成像。所以看到的仍然是 **蜡烛A完整的像**，只是像会变暗。故选 **C**。

*   **题目意思:** 用“F”字样探究平面镜成像特点。
*   **题目问题:** (1) 玻璃板如何放置？ (2) 选哪个物体b？结论？ (3) 探究像大小与距离关系结论？ (4) 玻璃板倾斜后会怎样？ (5) 哪种操作能改变像的位置？

    *   (1) 为了保证像和物能在同一水平面（白纸）上比较，玻璃板必须与白纸 **垂直** 放置。
    *   (2) 实验是为了比较像与物的大小关系，所以物体b必须与物体a大小相同。同时，平面镜成的像是左右相反的，所以要选择左右对称的②号“F”字样，才能与a的像完全重合。由此得出结论：像与物的大小 **相等**。

        *   B. 平面镜向上平移，物体不动，水平方向的相对位置不变，像的水平位置不变（只在竖直方向移动）。
        *   C. 绕左侧边转动，镜面方向改变，像的位置改变。
        *   D. 绕底边向后倾斜，镜面方向改变，像的位置改变。
