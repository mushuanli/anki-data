
# 高中数学二级结论 - 圆锥曲线篇

## 一. 椭圆

**结论1：** 椭圆两焦点为 $F_1, F_2$。P是椭圆上任意一点，则有以下结论成立：
1.  $|PF_1|+|PF_2|=2a$
2.  $a-c \le |PF_1| \le a+c$
3.  $b^2 \le |PF_1| \cdot |PF_2| \le a^2$
4.  焦半径公式: $|PF_1| = a+ex_0$, $|PF_2| = a-ex_0$

**结论2：**
1.  与椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 共焦点的椭圆方程可设为 $\frac{x^2}{a^2+\lambda} + \frac{y^2}{b^2+\lambda} = 1, (b^2+\lambda>0)$
2.  与椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 有相同离心率的椭圆可设为 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = \lambda, (\lambda>0)$

**结论3：** 椭圆方程为 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1 (a>b>0)$，两焦点为 $F_1, F_2$. $P(x_0, y_0)$是椭圆上任意一点，
1.  $y_0^2 = \frac{b^2}{a^2}(a^2-x_0^2)$, $x_0^2 = \frac{a^2}{b^2}(b^2-y_0^2)$
2.  参数方程 $\begin{cases} x_0 = a \cos\theta \\ y_0 = b \sin\theta \end{cases}$ ($\theta$为参数)

**结论4：** 若 $P(x_0, y_0)$在椭圆 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$上，则:
1.  以$P(x_0, y_0)$为切点的切线斜率为 $k = -\frac{b^2x_0}{a^2y_0}$
2.  过点P的椭圆的切线方程为 $\frac{x_0x}{a^2} + \frac{y_0y}{b^2} = 1$

**结论5：** 设P是椭圆上异于长轴端点的任意一点，两焦点为 $F_1, F_2$, 记$\angle F_1PF_2=\theta$，则：
1.  $|PF_1||PF_2| = \frac{2b^2}{1+\cos\theta}$
2.  焦点三角形面积 $S_{\triangle F_1PF_2} = c|y_p| = b^2\tan\frac{\theta}{2}$
3.  当点P位于短轴顶点处时, $\theta$最大. 此时 $S_{\triangle F_1PF_2}$ 也最大.
4.  $\cos\theta = 1-2e^2$
5.  点M是$\triangle F_1PF_2$的内心, PM交$F_1F_2$于点N, 则 $\frac{|PN|}{|NM|} = \frac{a}{c}$.

**结论6：** 若$P(x_0, y_0)$在椭圆 $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$外, 过点$P(x_0, y_0)$作椭圆的两条切线切点为$P_1, P_2$. 则切点弦$P_1P_2$的直线方程是 $\frac{x_0x}{a^2}+\frac{y_0y}{b^2}=1$.

**结论7：** 椭圆的两个顶点$A_1(-a,0), A_2(a,0)$, 与y轴平行的直线交椭圆于$P_1, P_2$时, $A_1P_1$与$A_2P_2$交点的轨迹方程是 $\frac{x^2}{a^2}-\frac{y^2}{b^2}=1$.

**结论8：** 椭圆上任意一点$A(x_0, y_0)$任意作两条倾斜角互补的直线交椭圆于B, C两点, 则直线BC的斜率$k_{BC}=-\frac{b^2x_0}{a^2y_0}$ (常数).

**结论9：** 设P是椭圆上异于长轴端点的任意一点, 两焦点为$F_1, F_2$, $\angle PF_1F_2=\alpha, \angle PF_2F_1=\beta$, 则 $e = \frac{c}{a} = \frac{\sin(\alpha+\beta)}{\sin\alpha+\sin\beta}$.

**结论10：** P为椭圆上任意一点, 两焦点为$F_1, F_2$. A为椭圆内一定点, 则 $2a-|AF_2| \le |PA|+|PF_1| \le 2a+|AF_1|$, 当且仅当A, F, P三点共线时等号成立.

**结论11：** 过焦点且垂直于长轴的弦叫通径, 其长度为$\frac{2b^2}{a}$.

**结论12：** 已知A, B是椭圆上的两点, 线段AB的垂直平分线与x轴交于点$P(x_0, 0)$, 则: $-\frac{a^2-b^2}{a} < x_0 < \frac{a^2-b^2}{a}$.

## 二. 双曲线

**结论1：**
1.  与 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 共轭的双曲线方程为 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = -1$.
    - ① 它们有公共渐近线
    - ② 四个焦点都在以原点为圆心, c为半径的圆上.
    - ③ $\frac{1}{e_1^2} + \frac{1}{e_2^2} = 1$.
2.  与 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 有相同焦点的双曲线方程为 $\frac{x^2}{a^2-\lambda} - \frac{y^2}{b^2+\lambda} = 1, (\lambda>0, a^2-\lambda>0, b^2+\lambda>0)$
3.  与 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 有相同焦点的椭圆方程为 $\frac{x^2}{a^2+\lambda} + \frac{y^2}{b^2+\lambda} = 1, (\lambda>0, a^2+\lambda>b^2+\lambda)$
4.  与 $\frac{x^2}{a^2} + \frac{y^2}{b^2} = 1$ 有相同焦点的双曲线方程为 $\frac{x^2}{a^2-\lambda} - \frac{y^2}{\lambda-b^2} = 1, (\lambda>0, a^2-\lambda>0, \lambda-b^2>0)$
5.  与 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 有相同离心率的双曲线方程为:
    - ① 焦点在x轴上时: $\frac{x^2}{a^2} - \frac{y^2}{b^2} = \lambda, (\lambda>0, \lambda \ne 1)$
    - ② 焦点在y轴上时: $\frac{x^2}{b^2} - \frac{y^2}{a^2} = \lambda, (\lambda>0)$
6.  与 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$ 有相同渐近线的双曲线方程为: $\frac{x^2}{a^2} - \frac{y^2}{b^2} = \lambda, (\lambda \ne 0, \lambda \ne 1)$

**结论2：** 双曲线的两焦点分别为 $F_1, F_2$. P是双曲线上任意一点, 则:
1.  $||PF_1|-|PF_2||=2a$
2.  $|PF_1|_{min}=a+c$, $|PF_1|_{min}=c-a$ (P在左支上)
    $|PF_2|_{min}=a+c$, $|PF_1|_{min}=c-a$ (P在右支上)

**结论3：** 双曲线的方程为 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1, (a>0, b>0)$. P($x_0, y_0$)是双曲线上任意一点, 则:
$y_0^2 = \frac{b^2}{a^2}(x_0^2-a^2)$, $x_0^2 = \frac{a^2}{b^2}(b^2+y_0^2)$.

**结论4：** 设P点是双曲线上异于实轴端点的任意一点, 两焦点分别为 $F_1, F_2$. 记 $\angle F_1PF_2 = \theta$. 则:
1.  $|PF_1| \cdot |PF_2| = \frac{2b^2}{1-\cos\theta}$
2.  焦点三角形面积 $S_{\triangle F_1PF_2} = c|y_p| = b^2|\cot\frac{\theta}{2}|$

**结论5：** 若 $P(x_0, y_0)$在双曲线 $\frac{x^2}{a^2} - \frac{y^2}{b^2} = 1$上, 则
1.  以$P(x_0, y_0)$为切点的切线斜率为 $k=\frac{b^2x_0}{a^2y_0}$
2.  过点P的双曲线的切线方程为 $\frac{x_0x}{a^2} - \frac{y_0y}{b^2} = 1$

**结论6：** 若$P(x_0, y_0)$在双曲线 $\frac{x^2}{a^2}-\frac{y^2}{b^2}=1$外, 则过P作双曲线的两条切线切点为$P_1, P_2$. 则切点弦$P_1P_2$的直线方程是 $\frac{x_0x}{a^2}-\frac{y_0y}{b^2}=1$.

**结论7：** 双曲线的两个顶点为$A_1(-a,0), A_2(a,0)$, 与y轴平行的直线交双曲线$P_1, P_2$时, $A_1P_1$与$A_2P_2$交点的轨迹方程为 $\frac{x^2}{a^2}+\frac{y^2}{b^2}=1$.

**结论8：** 过双曲线上任意一点 $A(x_0, y_0)$ 任意作两条倾斜角互补的直线交双曲线于B, C两点, 则直线BC的斜率$k_{BC}=\frac{b^2x_0}{a^2y_0}$ (常数).

**结论9：** 过焦点且垂直于长轴的弦叫通径, 其长度为$\frac{2b^2}{a}$.

**结论10：** 双曲线顶点到渐近线的距离是b. 焦点到渐近线的距离是b.

**结论11：** 双曲线实轴端点到两渐近线的距离之积为值是 $\frac{a^2b^2}{c^2}$.

**结论12：** 双曲线 $x^2-y^2=\pm a^2$ 称为等轴双曲线, 其渐近线方程为$y=\pm x$. 离心率$e=\sqrt{2}$.

**结论13：** 设双曲线 $\frac{x^2}{a^2}-\frac{y^2}{b^2}=1$, 其中两焦点坐标为$F_1(-c,0), F_2(c,0)$, 过$F_2$的直线L的倾斜角为$\alpha$, 交双曲线于A,B两点. 焦点在x轴的弦长为:
$|AB| = \begin{cases} \frac{2ab^2}{a^2-c^2\cos^2\alpha} & \text{(A,B在同一支双曲线上)} \\ \frac{2ab^2}{c^2\cos^2\alpha-a^2} & \text{(A,B在两支双曲线上)} \end{cases}$

## 三. 抛物线

**结论1：** 设AB为过抛物线 $y^2=2px (p>0)$ 焦点的弦. A$(x_1, y_1)$, B$(x_2, y_2)$. 直线AB的倾斜角为$\theta$. 则:
1.  $x_1x_2 = \frac{p^2}{4}$, $y_1y_2 = -p^2$
2.  $|AF| = x_1+\frac{p}{2} = \frac{p}{1-\cos\theta}$, $|BF| = x_2+\frac{p}{2} = \frac{p}{1+\cos\theta}$
3.  $|AB| = x_1+x_2+p = \frac{2p}{\sin^2\theta}$
4.  $\frac{1}{|FA|} + \frac{1}{|FB|} = \frac{2}{p}$
5.  $OA \cdot OB = -\frac{3}{4}p^2$
6.  $S_{\triangle AOB} = \frac{1}{2}|OA| \cdot |OB|\sin\angle AOB = \frac{1}{2}|OF| \cdot h_? = \frac{p^2}{2\sin\theta}$
7.  以AB为直径的圆与准线相切. 以AF为直径的圆与y轴相切.

**结论2：** 如图所示, 以A, B两点为切点引抛物线的两条切线, 两条切线交于一点M, 则有:
1.  M在准线上
2.  设线段AB中点为N, 则MN // x轴. 即$y_M = \frac{y_1+y_2}{2}$.
3.  $MF \perp AB$.

**结论3：** AB的中垂线与x轴交于点R, 则 $|AB|=2|FR|$.

**结论4：** 以A为切点的切线斜率为$\frac{p}{y_1}$, 切线方程为$y_1y = p(x+x_1)$.

**结论5：** 已知抛物线方程为$y^2=2px (p>0)$, 定点$M(m,0) (m \ne 0)$, 直线L过点M交抛物线于A, B两点, $A(x_1, y_1), B(x_2, y_2)$, 则有$x_1x_2=m^2$, $y_1y_2=-2pm$.

**结论6：** 已知A, B是抛物线$y^2=2px (p>0)$两点, 且直线AB不垂直于x轴. 则有: $k_{AB} = \frac{2p}{y_1+y_2}$. ($y_M$为线段AB中点纵坐标).

**结论7：**
*   抛物线$y^2=2px$的参数方程为 $\begin{cases} x = 2pt^2 \\ y = 2pt \end{cases}$ (t为参数).
*   抛物线$x^2=2py$的参数方程为 $\begin{cases} x = 2pt \\ y = 2pt^2 \end{cases}$ (t为参数).

**结论8：** 抛物线的中点弦问题:
1.  设抛物线$y^2=2px (p>0)$的弦为AB, 该弦的中点为$C(x_0, y_0)$, 则$k_{AB} = \frac{p}{y_0}$.
2.  设抛物线$y^2=-2px (p>0)$的弦为AB, 该弦的中点为$C(x_0, y_0)$, 则$k_{AB} = -\frac{p}{y_0}$.
3.  设抛物线$x^2=2py (p>0)$的弦为AB, 该弦的中点为$C(x_0, y_0)$, 则$k_{AB} = \frac{x_0}{p}$.
4.  设抛物线$x^2=-2py (p>0)$的弦为AB, 该弦的中点为$C(x_0, y_0)$, 则$k_{AB} = -\frac{x_0}{p}$.

**结论9：** 过抛物线$y^2=2px (p>0)$内一点$P(x_0, y_0)$, 作抛物线弦AB, 过A, B作抛物线的切线, 两条切线的交点M的轨迹方程为 $y_0y = p(x+x_0)$.

**结论10：** 抛物线$y^2=2px (p>0)$对称轴上一定点$A(a,0)$, 则:
1.  $0 < a \le p$时, 焦点到点A距离最小, 最小值为a.
2.  当$a > p$时, 抛物线上有关于x轴对称的两点到点A距离最小, 最小值为$\sqrt{2ap-p^2}$.

**结论11：** 抛物线$y^2=2px(p>0)$与直线$y=kx+b$ 相交于$A(x_1, y_1), B(x_2, y_2)$, 且该直线与y轴交于点$C(0, y_3)$, 则有 $\frac{1}{y_1}+\frac{1}{y_2}=\frac{1}{y_3}$.
